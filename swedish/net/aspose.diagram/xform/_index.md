## XForm class

Innehåller element som styr linjeattribut för en form, såsom mönster, vikt och färg. Dessa element avgör om linjeändarna är formaterade (till exempel med en pilspets), storleken på linjeslutsformaten, radien för den avrundade cirkeln som appliceras på linjen och linjekapselstilen (rund eller kvadratisk).

public class XForm : ICloneable
