## StreamProviderOptions.CustomPath property

Användarens anpassade sökväg (URL) sparad i genererad html-fil för den refererade källan. Om det inte har definierats av användaren kommer DefaultPath att användas. Till exempel kommer arkdata att sparas av användaren till d:/sheet001.htm, webbadressen som används i html-huvudfilen ska vara "d:/sheet001.htm" eller annan giltig relativ sökväg som kan nås av html-huvudfilen.

```csharp
public string CustomPath { set; }
```
