## PdfSaveOptions.SaveForegroundPagesOnly property

Anger om alla sidor ska sparas i bild eller endast i förgrunden.

```csharp
public bool SaveForegroundPagesOnly { get; set; }
```

### Anmärkningar

If true - renderade endast förgrundssidor (med bakgrund om det finns). If false - renderade förgrundssidor (med bakgrund om närvarande) efter det tomma bakgrundssidor. Kan endast returnera sant när PageCount &gt; 1. Standardvärdet är false .
