## PrintSaveOptions.SaveForegroundPagesOnly property

Anger om alla sidor ska skrivas ut eller bara förgrunden.

```csharp
public bool SaveForegroundPagesOnly { get; set; }
```

### Anmärkningar

Om sant - skrivs endast ut förgrundssidor (med bakgrund om det finns). Om falskt - utskrivna förgrundssidor (med bakgrund om det finns) efter det tomma bakgrundssidor. Kan endast returnera sant när PageCount &gt; 1. Standardvärdet är falskt .
