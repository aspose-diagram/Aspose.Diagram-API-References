## ImageSaveOptions.JpegQuality property

Hämtar eller ställer in ett värde som bestämmer kvaliteten på de genererade JPEG-bilderna.

```csharp
public int JpegQuality { get; set; }
```

### Anmärkningar

Har endast effekt när du sparar till JPEG. Använd den här egenskapen för att få eller ställa in kvaliteten på genererade bilder när du sparar i JPEG-format. Värdet kan variera från 0 till 100 där 0 betyder sämsta kvalitet men maximal komprimering och 100 betyder bäst kvalitet men minsta komprimering. Standardvärdet är 95.
