## HTMLSaveOptions.SaveAsSingleFile property

Indikerar om HTML-filen sparas som en enda fil. Standardvärdet är false.

```csharp
public bool SaveAsSingleFile { get; set; }
```

### Anmärkningar

Om det finns flera sidor måste dessa sidor och andra resurser sparas i separata filer. För vissa scenarier kanske användaren bara behöver få en resulterande fil, t.ex. för att underlätta överföringen. Om så är fallet kan användaren ställa in den här egenskapen som true.
