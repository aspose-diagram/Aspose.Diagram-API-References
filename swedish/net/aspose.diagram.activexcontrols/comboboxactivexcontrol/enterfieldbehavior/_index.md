## ComboBoxActiveXControl.EnterFieldBehavior property

Anger valbeteende när kontrollen går in. True anger att urvalet förblir oförändrat från senaste gången kontrollen var aktiv. False anger att all text i kontrollen kommer att väljas när du går in i kontrollen.
