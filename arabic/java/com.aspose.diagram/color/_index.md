

ينشئ كائن Color من قيم الألوان 8-بت المحددة (أحمر، أخضر، أزرق). قيمة ألفا هي ضمنيًا 255 (معتمة تمامًا). على الرغم من أن هذه الطريقة تسمح بتمرير قيمة 32-بت لكل مكوّن لون، فإن قيمة كل مكوّن محدودة بـ 8 بت.

**Parameters:**
| معامل | نوع | الوصف |
| --- | --- | --- |
| red | int | قيمة مكوّن الأحمر للكائن Color الجديد. القيم الصالحة هي من 0 إلى 255. |
| أخضر | int | قيمة المكوّن الأخضر لكائن Color الجديد. القيم الصالحة هي من 0 إلى 255. |
| أزرق | int | قيمة المكوّن الأزرق لكائن Color الجديد. القيم الصالحة هي من 0 إلى 255. |

**Returns:**
[Color](../../com.aspose.diagram/color) - The Color object that this method creates.

| --- | --- | --- |
| ألفا | int | المكوّن ألفا. القيم الصالحة هي من 0 إلى 255. |
| red | int | المكوّن الأحمر. القيم الصالحة هي من 0 إلى 255. |
| أخضر | int | المكوّن الأخضر. القيم الصالحة هي من 0 إلى 255. |
| أزرق | int | المكوّن الأزرق. القيم الصالحة هي من 0 إلى 255. |
