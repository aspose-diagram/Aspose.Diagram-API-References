### NO_ADV_CONFIG {#NO-ADV-CONFIG}
```
public static final int NO_ADV_CONFIG
```


يحد من سيطرة المستخدمين على طريقة تحديث مجموعة سجلات البيانات في مربع حوار تكوين التحديث لمجموعة سجلات البيانات. على وجه الخصوص، لا يمكن للمستخدمين تغيير المفتاح الأساسي أو تحديد متى يجب استبدال بيانات الشكل؛ ومع ذلك، يمكن للمستخدمين ضبط فترة التحديث وتغيير مصدر البيانات.
