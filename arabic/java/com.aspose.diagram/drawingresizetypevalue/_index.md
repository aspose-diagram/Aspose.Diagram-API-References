### DEPENDS_ON_DRAWING_SIZE_TYPE {#DEPENDS-ON-DRAWING-SIZE-TYPE}
```
public static final int DEPENDS_ON_DRAWING_SIZE_TYPE
```


ما إذا كانت الصفحة تُعيد تغيير حجمها تلقائيًا يعتمد على قيمة الخلية DrawingSizeType. إذا كان DrawingSizeType = 0 (حجم صفحة الرسم هو نفسه حجم الصفحة المطبوعة)، فإن الصفحة تُعيد تغيير حجمها تلقائيًا. إذا كانت قيمة DrawingSizeType أي شيء غير الصفر (0)، فإن الصفحة لا تُعيد تغيير حجمها تلقائيًا.
