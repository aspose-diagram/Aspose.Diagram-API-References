Определяет регистр текста формы. Все заглавные (uppercase) буквы (1) и начальные заглавные буквы (2) не меняют внешний вид текста, введённого полностью заглавными буквами. Текст должен быть введён строчными буквами, чтобы эти параметры проявились.
## Поля

| Поле | Описание |
| --- | --- |
| [ALL_CAPITAL_LETTERS](#ALL-CAPITAL-LETTERS) | Все заглавные (верхнего регистра) буквы. |
| [INITIAL_CAPITAL_LETTERS_ONLY](#INITIAL-CAPITAL-LETTERS-ONLY) | Только начальные заглавные буквы. |
| [NORMAL_CASE](#NORMAL-CASE) | Обычный регистр. |
| [UNDEFINED](#UNDEFINED) | Неопределено. |
