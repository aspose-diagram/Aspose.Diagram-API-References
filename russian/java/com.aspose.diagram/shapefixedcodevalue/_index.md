## Поля

| --- | --- |
| [ALLOW_ROUTING_TO_SIDES_WITH_CONNECTION_POINTS](#ALLOW-ROUTING-TO-SIDES-WITH-CONNECTION-POINTS) | Разрешать маршрутизацию только к сторонам с точками подключения. |
| [IGNORE_CONNECTION_POINT](#IGNORE-CONNECTION-POINT) | Игнорировать расположение точек подключения при маршрутизации к ним. |
| [NO_GLUE_TO_PERIMETER](#NO-GLUE-TO-PERIMETER) | Не привязывать к периметру этой фигуры. |
| [NO_MOVE_ALLOW_SHAPES_PLACED](#NO-MOVE-ALLOW-SHAPES-PLACED) | Не перемещать эту фигуру, но разрешать размещать на ней другие размещаемые фигуры. |
| [NO_MOVE_AND_NO_ALLOW_SHAPES_PLACED](#NO-MOVE-AND-NO-ALLOW-SHAPES-PLACED) | Не перемещать эту фигуру и не разрешать размещать на ней другие размещаемые фигуры. |
| [NO_MOVE_USING_LAY_OUT_SHAPES](#NO-MOVE-USING-LAY-OUT-SHAPES) | Не перемещать эту фигуру при использовании команды «Lay Out Shapes» в Microsoft Visio. |
| [UNDEFINED](#UNDEFINED) | Неопределено. |
