### getFormula() {#getFormula--}
```
public String getFormula()
```


Представляет формулу элемента. Этот атрибут может содержать одну из следующих строк: "someFormula", если формула существует локально, "No Formula", если формула локально удалена или заблокирована, или "Inh", если формула унаследована. Если атрибут отсутствует, формула элемента представляет собой простую константу.
