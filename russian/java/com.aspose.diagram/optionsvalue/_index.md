### NO_ADV_CONFIG {#NO-ADV-CONFIG}
```
public static final int NO_ADV_CONFIG
```


Ограничивает возможности пользователей контролировать, как обновляется набор данных, в диалоговом окне «Настроить обновление» для набора данных. В частности, пользователи не могут изменять первичный ключ или указывать, когда данные фигур должны быть перезаписаны; однако пользователи могут установить интервал обновления и могут изменить источник данных.
