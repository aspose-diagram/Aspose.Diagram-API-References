

Указывает, как указанный элемент управления будет отображать значения Null.

| ------- | ----------------------------------------------------------------------------------------------------------------------------------------------- |
| Setting | Description                                                                                                                                     |
| True    | Элемент управления будет переключаться между состояниями Да, Нет и Null. Элемент управления отображается затемнённым (серым), когда его свойство Value установлено в Null. |
| False   | (Default) Элемент управления будет переключаться между состояниями Да и Нет. Значения Null отображаются как значения Нет.                           |
