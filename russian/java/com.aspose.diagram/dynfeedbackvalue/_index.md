java.lang.Object
```
public final class DynFeedbackValue
```

Указывает тип визуальной обратной связи, предоставляемой пользователям при перетаскивании соединителя. Когда кнопка мыши отпускается, полученная форма соединителя не зависит от этой настройки. Этот элемент не применяется к маршрутизируемым соединителям.
## Поля

| Поле | Описание |
| --- | --- |
| [REMAIN_STRAIGHT](#REMAIN-STRAIGHT) | Оставайтесь прямыми (без ног). |
| [SHOW_FIVE_LEGS](#SHOW-FIVE-LEGS) | Показывать пять ног при перетаскивании. |
| [SHOW_THREE_LEGS](#SHOW-THREE-LEGS) | Показывать три ноги при перетаскивании. |
| [UNDEFINED](#UNDEFINED) | Неопределено. |
