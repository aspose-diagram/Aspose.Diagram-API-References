| [LEFT_THEN_DOWN](#LEFT-THEN-DOWN) | Сначала размещает формы влево, затем вниз. |

| [RIGHT_THEN_DOWN](#RIGHT-THEN-DOWN) | Сначала размещает формы вправо, затем вниз. |
| [RIGHT_TO_LEFT](#RIGHT-TO-LEFT) | Размещает форму справа налево. |
| [TOP_TO_BOTTOM](#TOP-TO-BOTTOM) | Размещает форму сверху вниз. |
