### getContextTypeLabel() {#getContextTypeLabel--}
```
public String getContextTypeLabel()
```


Если значение ContextType равно 2 или 3, этот атрибут обязателен для определения сравнения. Для ContextType = 2 параметр ContextTypeLabel должен быть меткой элемента данных фигуры, а если ContextType = 3, то ContextTypeLabel должен быть именем локальной строки.
