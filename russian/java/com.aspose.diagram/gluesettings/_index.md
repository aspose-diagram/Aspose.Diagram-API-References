Битовые значения указывают, включена ли конкретная настройка клея или выключена. Значение может быть суммой этих значений:
## Поля

| Поле | Описание |
| --- | --- |
| [CONNECTION_POINTS](#CONNECTION-POINTS) | Клей к точкам соединения. |
| [DISABLED](#DISABLED) | Клей отключен |
| [GEOMETRY](#GEOMETRY) | Клей к геометрии. |
| [GUIDES](#GUIDES) | Клей к направляющим. |
| [HANDLES](#HANDLES) | Клей к ручкам. |
| [NONE](#NONE) | Клей включён, но другие настройки клея не включены. |
| [UNDEFINED](#UNDEFINED) | Неопределено. |
| [VERTICES](#VERTICES) | Клей к вершинам. |
