Указывает, как размещаемые фигуры переворачиваются и/или вращаются на странице, когда фигуры размещаются с помощью команды «Lay Out Shapes» в Microsoft Visio. Допустимы следующие шестнадцатеричные значения.
## Поля

| [DEFAULT_NO_FLIP](#DEFAULT-NO-FLIP) | По умолчанию. |

| [FLIP_HORIZONTAL](#FLIP-HORIZONTAL) | Горизонтальное отражение. |
| [FLIP_VERTICAL](#FLIP-VERTICAL) | Вертикальное отражение. |
| [NO_FLIP](#NO-FLIP) | Без отражения. |
| [UNDEFINED](#UNDEFINED) | Неопределено. |
