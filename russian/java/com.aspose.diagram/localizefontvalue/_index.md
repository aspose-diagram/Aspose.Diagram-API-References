Указывает, следует ли локализовать (перевести на другой язык) текст фигуры.
## Поля

| Поле | Описание |
| --- | --- |
| [ALWAYS_LOCALIZE_FONT](#ALWAYS-LOCALIZE-FONT) | Всегда локализовать шрифт. |
| [LOCALIZE_FONT_ONLY_ARIAL_SYMBOL](#LOCALIZE-FONT-ONLY-ARIAL-SYMBOL) | Локализовать шрифт только если он Arial или Symbol (по умолчанию). |
| [NEVER_LOCALIZE_FONT](#NEVER-LOCALIZE-FONT) | Никогда не локализовать шрифт. |
| [UNDEFINED](#UNDEFINED) | Неопределено. |
