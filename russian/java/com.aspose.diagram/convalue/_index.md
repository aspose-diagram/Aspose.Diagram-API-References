Указывает тип поведения координаты x или y управляющего рычага после перемещения рычага.
## Поля

| Поле | Описание |
| --- | --- |
| [OFFSET_FROM_CENTER](#OFFSET-FROM-CENTER) | Смещение от центра. |
| [OFFSET_FROM_CENTER_HIDDEN](#OFFSET-FROM-CENTER-HIDDEN) | Смещение от центра, скрыто. |
| [OFFSET_FROM_LEFT_EDGE](#OFFSET-FROM-LEFT-EDGE) | Смещение от левого края. |
| [OFFSET_FROM_LEFT_EDGE_HIDDEN](#OFFSET-FROM-LEFT-EDGE-HIDDEN) | Смещение от левого края, скрыто. |
| [OFFSET_FROM_RIGHT_EDGE](#OFFSET-FROM-RIGHT-EDGE) | Смещение от правого края. |
| [OFFSET_FROM_RIGHT_EDGE_HIDDEN](#OFFSET-FROM-RIGHT-EDGE-HIDDEN) | Смещение от правого края, скрыто. |
| [PROPORTIONAL](#PROPORTIONAL) | Пропорционально. |
| [PROPORTIONAL_HIDDEN](#PROPORTIONAL-HIDDEN) | Пропорционально, скрыто. То же, что 0, но управляющая ручка не видна. |
| [PROPORTIONAL_LOCKED](#PROPORTIONAL-LOCKED) | Пропорционально заблокировано. |
| [PROPORTIONAL_LOCKED_HIDDEN](#PROPORTIONAL-LOCKED-HIDDEN) | Пропорционально заблокировано, скрыто. |
| [UNDEFINED](#UNDEFINED) | Неопределено. |
