Указывает, разрешён ли динамический (фигура‑к‑фигуре) клей при соединении с фигурой.
## Поля

| Поле | Описание |
| --- | --- |
| [ALLOW_DYNAMIC_GLUE](#ALLOW-DYNAMIC-GLUE) | Разрешить динамическое склеивание. |
| [ALLOW_DYNAMIC_GLUE_2002](#ALLOW-DYNAMIC-GLUE-2002) | Разрешить динамическое склеивание (устарело в Microsoft Visio 2002). |
| [ALLOW_DYNAMIC_GLUE_FOR_DYNAMIC_CONNECTOR](#ALLOW-DYNAMIC-GLUE-FOR-DYNAMIC-CONNECTOR) | По умолчанию. |
| [NO_ALLOW_2_D_SHAPE](#NO-ALLOW-2-D-SHAPE) | Запретить соединять эту 2‑D фигуру с помощью динамического склеивания. |
| [NO_ALLOW_DYNAMIC_GLUE](#NO-ALLOW-DYNAMIC-GLUE) | Запретить динамическое склеивание. |
| [UNDEFINED](#UNDEFINED) | Неопределено. |
