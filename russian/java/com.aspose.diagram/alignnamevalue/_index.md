| Поле | Описание |
| --- | --- |
| [ALIGN_TEXT_CENTER](#ALIGN-TEXT-CENTER) | Выровнять текст по центру. |

| [ALIGN_TEXT_RIGHT](#ALIGN-TEXT-RIGHT) | Выровнять текст справа. |
