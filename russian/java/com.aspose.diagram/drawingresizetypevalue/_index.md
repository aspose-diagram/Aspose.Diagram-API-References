### AUTOMATICALLY {#AUTOMATICALLY}
```
public static final int AUTOMATICALLY
```


Страница чертежа автоматически изменяется в размере.

### DEPENDS_ON_DRAWING_SIZE_TYPE {#DEPENDS-ON-DRAWING-SIZE-TYPE}
```
public static final int DEPENDS_ON_DRAWING_SIZE_TYPE
```


Автоматическое изменение размера страницы зависит от значения ячейки DrawingSizeType. Если DrawingSizeType = 0 (размер страницы чертежа совпадает с размером печатной страницы), страница изменяется автоматически. Если значение DrawingSizeType отличается от нуля (0), страница не изменяется автоматически.

public static final int NOT_AUTOMATICALLY
