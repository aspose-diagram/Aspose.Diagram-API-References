## HTMLSaveOptions.SaveAsSingleFile property

Указывает, следует ли сохранять html как один файл. Значение по умолчанию — false.

```csharp
public bool SaveAsSingleFile { get; set; }
```

### Примечания

Если страниц несколько, эти страницы и другие ресурсы необходимо сохранить в отдельные файлы. В некоторых сценариях пользователю может потребоваться получить только один результирующий файл, например, для удобства передачи. Если это так, пользователь может установить для этого свойства значение true.
