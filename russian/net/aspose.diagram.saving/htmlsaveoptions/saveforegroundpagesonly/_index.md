## HTMLSaveOptions.SaveForegroundPagesOnly property

Указывает, будут ли сохранены все страницы в изображении или только передний план.

```csharp
public bool SaveForegroundPagesOnly { get; set; }
```

### Примечания

Если true — отображаются только страницы переднего плана (с фоном, если он присутствует). Если false — отображаются страницы переднего плана (с фоном, если он есть) после этих пустых фоновых страниц. Может возвращать значение true, только когда PageCount &gt; 1. Значение по умолчанию — false .
