### Примечания

Метафайлы EMF, обозначенные как «EMF+ Dual», могут содержать как записи EMF+, так и записи EMF. Для рендеринга изображения можно использовать записи любого типа, только записи EMF+ или только записи EMF. Если установлено значение EmfPlusPrefer, будут анализироваться записи EMF+, в противном случае будут анализироваться только записи EMF. Значение по умолчанию — EmfOnly"/ &gt;.
