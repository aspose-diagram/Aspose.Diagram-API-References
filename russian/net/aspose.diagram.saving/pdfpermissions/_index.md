### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| DisallowAll | `0` | Запрещает все операции с документом PDF. Это значение по умолчанию. |
| Printing | `4` | Позволяет распечатать документ. |
| ModifyContents | `8` | Позволяет изменять содержимое документа. |
| ContentCopy | `10` | Позволяет копировать или иным образом извлекать текст и графику из документа, включая извлечение в целях доступности. |
| ModifyAnnotations | `20` | Позволяет добавлять или изменять текстовые аннотации. При использовании 40-битного шифрования RC4 эта опция также позволяет заполнять поля формы. |
| FillIn | `100` | Позволяет заполнять формы и подписывать документ. При использовании 40-битного шифрования RC4 этот параметр игнорируется, и заполнение формы разрешено всякий раз, когда установлен параметр ModifyAnnotations. |
| ContentCopyForAccessibility | `200` | Позволяет извлекать текст и графику для обеспечения доступности для пользователей с ограниченными возможностями или для других целей. При использовании 40-битного шифрования RC4 этот параметр игнорируется, а специальные возможности разрешаются при установке ContentCopy. |
| DocumentAssembly | `400` | Позволяет собирать документ: вставлять, поворачивать или удалять страницы и создавать элементы навигации, такие как закладки или эскизы изображений. При использовании 40-битного шифрования RC4 этот параметр игнорируется, и сборка документа разрешена, если установлен параметр ModifyContents. |
| HighResolutionPrinting | `804` | Позволяет печатать документ с максимально возможным разрешением. При использовании 40-битного шифрования RC4 этот параметр игнорируется, и разрешена печать с высоким разрешением, если задано значение «Печать». |
| AllowAll | `FFFF` | Разрешает все операции с документом PDF. |
