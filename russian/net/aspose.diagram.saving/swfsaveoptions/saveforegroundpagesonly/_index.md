## SWFSaveOptions.SaveForegroundPagesOnly property

Указывает, будут ли все страницы сохранены в изображении или только передний план.

```csharp
public bool SaveForegroundPagesOnly { get; set; }
```

### Примечания

Если true - отображаются только страницы переднего плана (с фоном, если он есть). Если false - отображаются страницы переднего плана (с фоном, если он присутствует) после этих пустых фоновых страниц. Может возвращать значение true, только если PageCount &gt; 1. Значение по умолчанию — false.
