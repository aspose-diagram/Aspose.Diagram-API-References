## PrintSaveOptions.SaveForegroundPagesOnly property

Указывает, будут ли печататься все страницы или только передний план.

```csharp
public bool SaveForegroundPagesOnly { get; set; }
```

### Примечания

Если true - печатаются только страницы переднего плана (с фоном, если он есть). Если false - печатаются страницы переднего плана (с фоном, если он есть) после этих пустых фоновых страниц. Может возвращать значение true, только если PageCount &gt; 1. Значение по умолчанию — false.
