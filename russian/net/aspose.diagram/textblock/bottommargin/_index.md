## TextBlock.BottomMargin property

Определяет расстояние между нижней границей текстового блока и последней строкой содержащегося в нем текста. По умолчанию 4 pt. Это значение не зависит от масштаба чертежа. Если чертеж масштабируется, нижнее поле остается прежним.

public DoubleValue BottomMargin { get; set; }
