## Master class

Содержит элементы, определяющие мастер документа. Мастер — это фигура на трафарете, которую вы неоднократно используете для создания рисунков. Когда вы перетаскиваете фигуру из набора элементов на страницу документа, она становится экземпляром этого образца, а локальная копия образца включается в документ.

```csharp
public class Master : IDisposable
```
