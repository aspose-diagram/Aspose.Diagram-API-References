## DocumentProperties.HyperlinkBase property

Содержит путь, который будет использоваться для относительных гиперссылок (гиперссылок, для которых расположение связанного файла описано относительно диаграммы Microsoft Visio). По умолчанию путь гиперссылки относится к текущему документу, если в этом элементе не указан другой путь. Максимальная длина 256 символов.

public string HyperlinkBase { get; set; }
