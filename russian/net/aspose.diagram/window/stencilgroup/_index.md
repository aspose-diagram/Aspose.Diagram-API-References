## Window.StencilGroup property

Указывает группу объединенных окон трафарета, членом которой является окно. Этот атрибут актуален только для элементов Window, чей атрибут WindowType имеет значение Stencil, и только в том случае, если окно шаблона является частью объединенной группы окон шаблона. Все окна трафарета, являющиеся частью одной и той же объединенной группы, имеют одинаковое значение элемента StencilGroup.

```csharp
public string StencilGroup { get; set; }
```
