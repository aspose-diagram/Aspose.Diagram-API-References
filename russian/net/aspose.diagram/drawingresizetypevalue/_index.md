## DrawingResizeTypeValue enumeration

Определяет, будет ли автоматически изменяться размер страницы документа в соответствии с диаграммой.

```csharp
public enum DrawingResizeTypeValue
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| DependsOnDrawingSizeType | `0` | Автоматическое изменение размера страницы зависит от значения ячейки DrawingSizeType. Если DrawingSizeType = 0 (размер страницы чертежа совпадает с размером напечатанной страницы), размер страницы изменяется автоматически. Если значение DrawingSizeType отлично от нуля (0), размер страницы не изменяется автоматически. |
| Automatically | `1` | Размер страницы документа изменяется автоматически. |
| NotAutomatically | `2` | Размер страницы чертежа не изменяется автоматически. |
| Undefined | `-2147483648` | Не определено. |
