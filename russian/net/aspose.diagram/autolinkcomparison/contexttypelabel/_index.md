## AutoLinkComparison.ContextTypeLabel property

Если значение ContextType равно 2 или 3, этот атрибут необходим для определения сравнения. Для ContextType = 2 ContextTypeLabel должен быть меткой элемента данных формы, а если ContextType = 3, ContextTypeLabel должен быть именем локальной строки.

public string ContextTypeLabel { get; set; }
