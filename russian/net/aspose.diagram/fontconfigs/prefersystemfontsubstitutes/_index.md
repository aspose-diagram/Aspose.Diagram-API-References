## FontConfigs.PreferSystemFontSubstitutes property

Указывает, следует ли сначала использовать заменители системного шрифта или нет, если шрифт не представлен и замена этого шрифта не установлена. Например, в Ubuntu шрифт "Arial" обычно заменяется на "Liberation Sans". Значение по умолчанию — false.

```csharp
public static bool PreferSystemFontSubstitutes { get; set; }
```
