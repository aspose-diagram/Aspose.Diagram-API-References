## RuleValue.Formula property

Представляет формулу элемента. Этот атрибут может содержать одну из следующих строк: «someFormula», если формула существует локально, «No Formula», если формула локально удалена или заблокирована, или «Inh», если формула унаследована. Если атрибут отсутствует, формула элемента является простой константой.

public string Formula { get; set; }
