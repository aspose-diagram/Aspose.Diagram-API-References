## DataRecordSet.RowOrder property

Следует ли использовать порядок строк в наборе записей данных в качестве первичного ключа. Истинно (1), если идентификаторы строк определяются порядком строк. False (0), если идентификаторы строк определяются значениями в столбцах первичного ключа набора записей данных.
