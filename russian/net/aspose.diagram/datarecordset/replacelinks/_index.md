## DataRecordSet.ReplaceLinks property

Определяет, как обрабатываются ссылки данных формы при копировании или вырезании фигур. 1, чтобы заменить существующие ссылки в целевой форме. 0, чтобы сохранить существующие ссылки в целевой форме. Если этот атрибут отсутствует, Visio спрашивает пользователя, заменять ли ссылки при копировании или вырезании.

public int ReplaceLinks { get; set; }
