## CheckBoxActiveXControl.IsTripleState property

Указывает, как указанный элемент управления будет отображать нулевые значения.

```csharp
public bool IsTripleState { get; set; }
```

### Примечания

**Параметр**

**Описание**

Истинный

Элемент управления будет циклически переключаться между состояниями для значений Да, Нет и Нуль. Элемент управления отображается затемненным (серым), когда для его свойства Value установлено значение Null.

ЛОЖЬ

(По умолчанию) Элемент управления будет циклически переключаться между состояниями для значений «Да» и «Нет». Значения Null отображаются так, как если бы они были No values.
