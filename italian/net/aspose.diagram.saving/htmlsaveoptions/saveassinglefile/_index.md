## HTMLSaveOptions.SaveAsSingleFile property

Indica se salvare l'html come singolo file. Il valore predefinito è false.

```csharp
public bool SaveAsSingleFile { get; set; }
```

### Osservazioni

Se ci sono più pagine, quelle pagine e altre risorse devono essere salvate in file separati. Per alcuni scenari, l'utente potrebbe aver bisogno di ottenere un solo file risultante, ad esempio per comodità di trasferimento. In tal caso, l'utente può impostare questa proprietà su true.
