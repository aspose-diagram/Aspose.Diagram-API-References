## PrintSaveOptions.SaveForegroundPagesOnly property

Specifica se verranno stampate tutte le pagine o solo in primo piano.

```csharp
public bool SaveForegroundPagesOnly { get; set; }
```

### Osservazioni

Se vero - stampa solo le pagine in primo piano (con sfondo se presente). Se falso - stampa le pagine in primo piano (con sfondo se presente) dopo le pagine di sfondo vuote. Può restituire vero solo quando PageCount &gt; 1. Il valore predefinito è falso .
