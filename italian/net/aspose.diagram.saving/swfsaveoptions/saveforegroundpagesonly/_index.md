## SWFSaveOptions.SaveForegroundPagesOnly property

Specifica se tutte le pagine verranno salvate nell'immagine o solo in primo piano.

```csharp
public bool SaveForegroundPagesOnly { get; set; }
```

### Osservazioni

Se vero - visualizza solo le pagine in primo piano (con sfondo se presente). Se falso - visualizza le pagine in primo piano (con sfondo se presente) dopo le pagine di sfondo vuote. Può restituire vero solo quando PageCount &gt; 1. Il valore predefinito è falso .
