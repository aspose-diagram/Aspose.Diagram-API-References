## ImageSaveOptions.JpegQuality property

Ottiene o imposta un valore che determina la qualità delle immagini JPEG generate.

```csharp
public int JpegQuality { get; set; }
```

### Osservazioni

Ha effetto solo quando si salva in JPEG. Utilizzare questa proprietà per ottenere o impostare la qualità delle immagini generate durante il salvataggio in formato JPEG. Il valore può variare da 0 a 100 dove 0 indica la qualità peggiore ma la compressione massima e 100 indica la migliore qualità ma compressione minima. Il valore predefinito è 95.
