## ForeignData.CompressionLevel property

Questo attributo è significativo solo se i dati esterni sono un oggetto esterno basato su raster, ad esempio un file DIB, JPG, PNG, TIFF o GIF. Il valore indica il livello di compressione applicato al file. Il livello di compressione è misurato in centinaia di percento.

public double CompressionLevel { get; set; }
