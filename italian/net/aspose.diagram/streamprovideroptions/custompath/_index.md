## StreamProviderOptions.CustomPath property

Il percorso personalizzato dell'utente (URL) salvato nel file html generato per la fonte indicata. Se non definito dall'utente, verrà utilizzato DefaultPath. Ad esempio, i dati del foglio verranno salvati dall'utente in d:/sheet001.htm, l'URL utilizzato nel file html principale dovrebbe essere "d:/sheet001.htm" o altro percorso relativo valido a cui è possibile accedere dal file html principale.
