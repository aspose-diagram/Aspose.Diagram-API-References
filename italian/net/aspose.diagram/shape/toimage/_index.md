## ToImage(string, ImageSaveOptions) {#toimage_1}

Crea l'immagine della forma e la salva in un file. L'estensione del nome del file determina il formato dell'immagine.

```csharp
public void ToImage(string imageFile, ImageSaveOptions options)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| imageFile | String | Il nome del file immagine con il percorso completo. |
| options | ImageSaveOptions | Ulteriori opzioni per la creazione di immagini |

### Osservazioni

Il formato dell'immagine viene specificato utilizzando l'estensione del nome del file. Ad esempio, se specifichi "myfile.png", l'immagine verrà salvata nel formato PNG. Sono riconosciute le seguenti estensioni di file: .bmp, .gif, .png, .jpg, .jpeg, .tiff, .tif, .emf.
