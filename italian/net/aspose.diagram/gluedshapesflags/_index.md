## GluedShapesFlags enumeration

Specifica le costanti che identificano quali forme restituire, in base alla dimensionalità e alla direzionalità dei punti di connessione che sono incollati a una particolare forma; passato al metodo Shape.GluedShapes.

```csharp
public enum GluedShapesFlags
```

### I valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| GluedShapesAll1D | `0` | Restituisce gli ID di tutte le forme 1D incollate a questa forma. |
| GluedShapesIncoming1D | `1` | Restituisce gli ID delle forme 1D i cui punti finali sono incollati a questa forma. |
| GluedShapesOutgoing1D | `2` | Restituisce gli ID delle forme 1D i cui punti iniziali sono incollati a questa forma. |
| GluedShapesAll2D | `3` | Restituisce tutte le forme 2D che sono incollate a questa forma e tutte le forme 2D a cui è incollata questa forma. |
| GluedShapesIncoming2D | `4` | Se l'oggetto di origine è una forma 1D, restituisce gli ID della forma 2D a cui è incollato il punto iniziale. Se l'oggetto di origine è una forma 2D, restituisce gli ID delle forme 2D che sono incollate a questa forma. |
| GluedShapesOutgoing2D | `5` | Se l'oggetto di origine è una forma 1D, restituisce gli ID della forma 2D a cui è incollato il punto finale. Se l'oggetto di origine è una forma 2D, restituisce gli ID delle forme 2D a cui è incollata questa forma. |
