## AutoLinkComparison.ContextTypeLabel property

Se il valore ContextType è 2 o 3, questo attributo è necessario per definire un confronto. Per ContextType = 2, ContextTypeLabel deve essere l'etichetta dell'elemento dati della forma e se ContextType = 3, ContextTypeLabel deve essere il nome della riga locale.
