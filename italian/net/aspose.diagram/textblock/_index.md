## TextBlock class

Contiene elementi che specificano l'allineamento, i margini e le posizioni di tabulazione predefinite del testo nel blocco di testo di una forma.

```csharp
public class TextBlock : ICloneable
```

| [BottomMargin](../../aspose.diagram/textblock/bottommargin/) { get; set; } | Determina la distanza tra il bordo inferiore del blocco di testo e l'ultima riga di testo che contiene.
