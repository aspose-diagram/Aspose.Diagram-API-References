## ComboBoxActiveXControl.EnterFieldBehavior property

컨트롤을 입력할 때 선택 동작을 지정합니다. True는 컨트롤이 마지막으로 활성화되었을 때 선택 항목이 변경되지 않은 상태로 유지되도록 지정합니다. False는 컨트롤에 들어갈 때 컨트롤의 모든 텍스트가 선택되도록 지정합니다.

public bool EnterFieldBehavior { get; set; }
