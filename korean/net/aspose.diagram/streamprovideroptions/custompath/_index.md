## StreamProviderOptions.CustomPath property

참조 소스에 대해 생성된 html 파일에 저장된 사용자 정의 경로(URL)입니다. 사용자가 정의하지 않은 경우 DefaultPath가 사용됩니다. 예를 들어 시트 데이터는 사용자가 d:/sheet001.htm에 저장하고 기본 html 파일에서 사용되는 URL은 "d:/sheet001.htm"이어야 합니다. 기본 html 파일에서 액세스할 수 있는 다른 유효한 상대 경로입니다.

```csharp
public string CustomPath { set; }
```
