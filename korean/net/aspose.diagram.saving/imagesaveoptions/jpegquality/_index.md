## ImageSaveOptions.JpegQuality property

생성된 JPEG 이미지의 품질을 결정하는 값을 가져오거나 설정합니다.

```csharp
public int JpegQuality { get; set; }
```

### 비고

JPEG로 저장할 때만 효과가 있습니다. JPEG 형식으로 저장할 때 생성된 이미지의 품질을 가져오거나 설정하려면 이 속성을 사용하세요. 값은 0에서 100까지 다양할 수 있습니다. 여기서 0은 품질이 가장 낮지만 최대 압축률을 의미하고 100은 최고를 의미합니다. 품질이지만 최소 압축률입니다. 기본값은 95입니다.
