## AutoLinkComparison.ContextTypeLabel property

Jika nilai ContextType adalah 2 atau 3, atribut ini diperlukan untuk menentukan perbandingan. Untuk ContextType = 2, ContextTypeLabel harus berupa label item data bentuk, dan jika ContextType = 3, ContextTypeLabel harus berupa nama baris lokal.

public string ContextTypeLabel { get; set; }
