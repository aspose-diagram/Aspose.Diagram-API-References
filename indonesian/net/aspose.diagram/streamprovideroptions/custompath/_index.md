## StreamProviderOptions.CustomPath property

Jalur khusus pengguna (URL) disimpan dalam file html yang dihasilkan untuk sumber yang dirujuk. Jika tidak ditentukan oleh pengguna, DefaultPath akan digunakan. Misalnya, data sheet akan disimpan oleh pengguna ke d:/sheet001.htm, url yang digunakan dalam file html utama harus "d:/sheet001.htm" atau jalur relatif valid lainnya yang dapat diakses oleh file html utama.

public string CustomPath { set; }
