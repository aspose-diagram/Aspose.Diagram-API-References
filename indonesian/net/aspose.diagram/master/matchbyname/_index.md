## Master.MatchByName property

Atribut MatchByName menentukan bagaimana Microsoft Visio memutuskan apakah master dokumen sudah ada saat instance master dijatuhkan di halaman gambar. Ini memungkinkan perubahan yang dilakukan pada master dokumen untuk diterapkan ke instance baru master, bahkan jika instance diseret dari file stensil yang berdiri sendiri.
