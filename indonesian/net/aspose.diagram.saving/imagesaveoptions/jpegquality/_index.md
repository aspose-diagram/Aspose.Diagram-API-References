## ImageSaveOptions.JpegQuality property

Mendapat atau menetapkan nilai yang menentukan kualitas gambar JPEG yang dihasilkan.

```csharp
public int JpegQuality { get; set; }
```

### Perkataan

Hanya berpengaruh saat menyimpan ke JPEG. Gunakan properti ini untuk mendapatkan atau menyetel kualitas gambar yang dihasilkan saat menyimpan dalam format JPEG. Nilai dapat bervariasi dari 0 hingga 100 di mana 0 berarti kualitas terburuk tetapi kompresi maksimum dan 100 berarti terbaik kualitas tetapi kompresi minimum. Nilai default adalah 95.
