public enum PdfPermissions

### Nilai

| Nama | Nilai | Keterangan |
| --- | --- | --- |
| DisallowAll | `0` | Melarang semua operasi pada dokumen PDF. Ini adalah nilai default. |
| Printing | `4` | Memungkinkan pencetakan dokumen. |
| ModifyContents | `8` | Memungkinkan modifikasi isi dokumen. |
| ContentCopy | `10` | Memungkinkan penyalinan atau mengekstrak teks dan gambar dari dokumen, termasuk ekstraksi untuk tujuan aksesibilitas. |
| ModifyAnnotations | `20` | Memungkinkan penambahan atau modifikasi anotasi teks. Saat menggunakan enkripsi RC4 40-bit, opsi ini juga memungkinkan pengisian kolom formulir. |
| FillIn | `100` | Memungkinkan mengisi formulir dan menandatangani dokumen. Saat menggunakan enkripsi RC4 40-bit, opsi ini diabaikan dan pengisian formulir diperbolehkan setiap kali ModifyAnnotations disetel. |
| ContentCopyForAccessibility | `200` | Memungkinkan ekstrak teks dan grafik untuk mendukung aksesibilitas bagi pengguna yang dinonaktifkan atau untuk tujuan lain. Saat menggunakan enkripsi RC4 40-bit, opsi ini diabaikan dan aksesibilitas diizinkan setiap kali ContentCopy disetel. |
| DocumentAssembly | `400` | Memungkinkan perakitan dokumen: menyisipkan, memutar, atau menghapus halaman dan membuat elemen navigasi seperti bookmark atau gambar mini. Saat menggunakan enkripsi 40-bit RC4, opsi ini diabaikan dan perakitan dokumen diizinkan saat ModifyContents diatur. |
| HighResolutionPrinting | `804` | Memungkinkan pencetakan dokumen dengan resolusi setinggi mungkin. Saat menggunakan enkripsi RC4 40-bit, opsi ini diabaikan dan pencetakan resolusi tinggi diperbolehkan saat Pencetakan diatur. |
| AllowAll | `FFFF` | Memungkinkan semua operasi pada dokumen PDF. |
