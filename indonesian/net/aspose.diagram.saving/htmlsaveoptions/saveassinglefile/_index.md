## HTMLSaveOptions.SaveAsSingleFile property

Menunjukkan apakah menyimpan html sebagai file tunggal. Nilai defaultnya salah.

```csharp
public bool SaveAsSingleFile { get; set; }
```

### Perkataan

Jika ada beberapa halaman, halaman tersebut dan sumber daya lainnya harus disimpan ke dalam file terpisah. Untuk beberapa skenario, pengguna mungkin hanya perlu mendapatkan satu file yang dihasilkan seperti untuk kemudahan transfer. Jika demikian, pengguna dapat menyetel properti ini sebagai true.
