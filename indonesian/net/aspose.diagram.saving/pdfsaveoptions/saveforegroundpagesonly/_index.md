## PdfSaveOptions.SaveForegroundPagesOnly property

Menentukan apakah semua halaman akan disimpan dalam gambar atau hanya latar depan.

```csharp
public bool SaveForegroundPagesOnly { get; set; }
```

### Perkataan

Jika benar - hanya merender halaman latar depan (dengan latar belakang jika ada). Jika salah - merender halaman depan (dengan latar belakang jika ada) setelah halaman latar kosong itu. Dapat mengembalikan benar hanya jika PageCount &gt; 1. Nilai defaultnya salah .
