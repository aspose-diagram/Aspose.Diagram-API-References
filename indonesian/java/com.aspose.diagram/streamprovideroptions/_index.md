### setCustomPath(String value) {#setCustomPath-java.lang.String-}
```
public void setCustomPath(String value)
```


Jalur khusus (URL) pengguna yang disimpan dalam file html yang dihasilkan untuk sumber yang dirujuk. Jika tidak ditentukan oleh pengguna, DefaultPath akan digunakan. Misalnya, data lembar akan disimpan oleh pengguna ke d:/sheet001.htm, URL yang digunakan dalam file html utama harus "d:/sheet001.htm" atau jalur relatif valid lainnya yang dapat diakses oleh file html utama.
