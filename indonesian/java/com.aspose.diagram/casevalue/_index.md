public final class CaseValue

Menentukan huruf pada teks shape. Semua huruf kapital (uppercase) (1) dan huruf kapital awal (2) tidak mengubah tampilan teks yang dimasukkan dengan semua huruf kapital. Teks harus dimasukkan dengan huruf kecil agar opsi-opsi ini berpengaruh.
## Bidang

| Bidang | Deskripsi |
| --- | --- |
| [ALL_CAPITAL_LETTERS](#ALL-CAPITAL-LETTERS) | Semua huruf kapital (uppercase). |
| [INITIAL_CAPITAL_LETTERS_ONLY](#INITIAL-CAPITAL-LETTERS-ONLY) | Hanya huruf kapital di awal. |
| [NORMAL_CASE](#NORMAL-CASE) | Huruf normal. |
| [UNDEFINED](#UNDEFINED) | Tidak terdefinisi. |
