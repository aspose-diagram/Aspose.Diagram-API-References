### DEPENDS_ON_DRAWING_SIZE_TYPE {#DEPENDS-ON-DRAWING-SIZE-TYPE}
```
public static final int DEPENDS_ON_DRAWING_SIZE_TYPE
```


Apakah halaman mengubah ukuran secara otomatis tergantung pada nilai sel DrawingSizeType. Jika DrawingSizeType = 0 (ukuran halaman gambar sama dengan ukuran halaman yang dicetak), halaman akan mengubah ukuran secara otomatis. Jika nilai DrawingSizeType bukan nol (0), halaman tidak mengubah ukuran secara otomatis.
