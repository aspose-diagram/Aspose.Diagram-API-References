Menentukan konstanta yang mengidentifikasi bentuk mana yang akan dikembalikan, berdasarkan dimensi dan arah titik koneksi yang dilem pada bentuk tertentu; diteruskan ke metode Shape.GluedShapes.
## Bidang

| Bidang | Deskripsi |
| --- | --- |
| [GLUED_SHAPES_ALL_1_D](#GLUED-SHAPES-ALL-1-D) | Kembalikan ID semua shape 1-D yang ditempelkan pada shape ini. |
| [GLUED_SHAPES_ALL_2_D](#GLUED-SHAPES-ALL-2-D) | Kembalikan semua shape 2-D yang ditempelkan pada shape ini dan semua shape 2-D yang menempel pada shape ini. |
| [GLUED_SHAPES_INCOMING_1_D](#GLUED-SHAPES-INCOMING-1-D) | Kembalikan ID shape 1-D yang titik ujungnya ditempelkan pada shape ini. |
| [GLUED_SHAPES_INCOMING_2_D](#GLUED-SHAPES-INCOMING-2-D) | Jika objek sumber adalah shape 1-D, kembalikan ID shape 2-D yang titik awalnya ditempelkan. |
| [GLUED_SHAPES_OUTGOING_1_D](#GLUED-SHAPES-OUTGOING-1-D) | Kembalikan ID bentuk 1-D yang titik awalnya ditempelkan pada bentuk ini. |
| [GLUED_SHAPES_OUTGOING_2_D](#GLUED-SHAPES-OUTGOING-2-D) | Jika objek sumber adalah bentuk 1-D, kembalikan ID bentuk 2-D yang titik akhirnya ditempelkan. |
