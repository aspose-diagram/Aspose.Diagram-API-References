## HTMLSaveOptions.SaveAsSingleFile property

Geeft aan of de html als enkel bestand moet worden opgeslagen. De standaardwaarde is false.

```csharp
public bool SaveAsSingleFile { get; set; }
```

### Opmerkingen

Als er meerdere pagina's zijn, moeten die pagina's en andere bronnen in afzonderlijke bestanden worden opgeslagen. Voor sommige scenario's hoeft de gebruiker misschien slechts één resulterend bestand op te halen, bijvoorbeeld voor het gemak van overdracht. Als dit het geval is, kan de gebruiker deze eigenschap instellen als waar.
