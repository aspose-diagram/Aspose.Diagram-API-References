## StreamProviderOptions.CustomPath property

Het aangepaste pad (URL) van de gebruiker dat is opgeslagen in het gegenereerde HTML-bestand voor de verwezen bron. Indien niet gedefinieerd door de gebruiker, wordt DefaultPath gebruikt. De bladgegevens worden bijvoorbeeld door de gebruiker opgeslagen in d:/sheet001.htm, de url die wordt gebruikt in het html-hoofdbestand moet "d:/sheet001.htm" zijn of ander geldig relatief pad dat toegankelijk is via het HTML-hoofdbestand.

public string CustomPath { set; }
