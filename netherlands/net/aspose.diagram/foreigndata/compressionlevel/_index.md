## ForeignData.CompressionLevel property

Dit attribuut is alleen zinvol als de vreemde gegevens een op rasters gebaseerd vreemd object zijn, zoals een DIB-, JPG-, PNG-, TIFF- of GIF-bestand. De waarde geeft het compressieniveau aan dat op het bestand is toegepast. Het compressieniveau wordt gemeten in honderden procenten.

public double CompressionLevel { get; set; }
