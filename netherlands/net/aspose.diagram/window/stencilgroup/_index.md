## Window.StencilGroup property

Specificeert de groep samengevoegde stencilvensters waarvan het venster lid is. Dit attribuut is alleen relevant voor Window-elementen waarvan het WindowType-attribuut Stencil is, en alleen als het stencilvenster deel uitmaakt van een samengevoegde groep stencilvensters. Alle stencilvensters die deel uitmaken van dezelfde samengevoegde groep hebben dezelfde waarde voor het StencilGroup-element.

```csharp
public string StencilGroup { get; set; }
```
