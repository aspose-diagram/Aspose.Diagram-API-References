java.lang.Object
```
public class EventItem
```

Omsluit een gebeurteniscode. Een EventItem-element kan twee soorten acties activeren: het kan een add-on uitvoeren, of het kan een melding van de gebeurtenis naar het aanroepende programma sturen.

### getAction() {#getAction--}
