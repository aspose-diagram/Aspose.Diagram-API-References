### setCustomPath(String value) {#setCustomPath-java.lang.String-}
```
public void setCustomPath(String value)
```


Het door de gebruiker aangepaste pad(URL) opgeslagen in het gegenereerde html‑bestand voor de genoemde bron. Indien niet door de gebruiker gedefinieerd, wordt DefaultPath gebruikt. Bijvoorbeeld, de sheet‑gegevens worden door de gebruiker opgeslagen naar d:/sheet001.htm, de URL die in het hoofd‑html‑bestand wordt gebruikt, moet "d:/sheet001.htm" zijn of een ander geldig relatief pad dat toegankelijk is voor het hoofd‑html‑bestand.
