Maakt een Color-object aan vanuit de opgegeven 8-bit kleurwaarden (rood, groen en blauw). De alfa-waarde is impliciet 255 (volledig ondoorzichtig). Hoewel deze methode toestaat dat een 32-bit waarde wordt doorgegeven voor elke kleurcomponent, is de waarde van elke component beperkt tot 8 bits.

**Parameters:**

| --- | --- | --- |
| rood | int | De rode componentwaarde voor het nieuwe Color-object. Geldige waarden zijn 0 tot en met 255. |
| groen | int | De groencomponentwaarde voor het nieuwe Color-object. Geldige waarden zijn 0 tot en met 255. |
| blauw | int | De blauwcomponentwaarde voor het nieuwe Color-object. Geldige waarden zijn 0 tot en met 255. |

[Color](../../com.aspose.diagram/color) - The Color object that this method creates.

| groen | int | De groencomponent. Geldige waarden zijn 0 tot en met 255. |
