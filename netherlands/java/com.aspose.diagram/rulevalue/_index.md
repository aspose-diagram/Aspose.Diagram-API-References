### getFormula() {#getFormula--}
```
public String getFormula()
```


Stelt de formule van het element voor. Dit attribuut kan een van de volgende strings bevatten: "someFormula" als de formule lokaal bestaat, "No Formula" als de formule lokaal is verwijderd of geblokkeerd, of "Inh" als de formule is geërfd. Als het attribuut niet aanwezig is, is de formule van het element een eenvoudige constante.
