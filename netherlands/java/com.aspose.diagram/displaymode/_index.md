**Inheritance:**
java.lang.Object
```
public class DisplayMode
```

Wanneer het zich bevindt in een Group element, specificeert het DisplayMode element hoe een groepsvorm en zijn leden worden weergegeven. Wanneer het zich bevindt in een SmartTagDef element, bepaalt het DisplayMode element of de smart tag verschijnt wanneer de gebruiker de muis boven de tag houdt, wanneer de vorm is geselecteerd, of altijd.
