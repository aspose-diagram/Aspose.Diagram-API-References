```
public String getContextTypeLabel()
```


Als de ContextType‑waarde 2 of 3 is, is dit attribuut vereist om een vergelijking te definiëren. Voor ContextType = 2 moet ContextTypeLabel het label van het vorm‑gegevenselement zijn, en voor ContextType = 3 moet ContextTypeLabel de lokale rijnaam zijn.
