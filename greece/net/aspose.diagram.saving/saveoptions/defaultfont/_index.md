## SaveOptions.DefaultFont property

Όταν οι χαρακτήρες στο διάγραμμα είναι unicode και δεν έχουν οριστεί με τη σωστή τιμή γραμματοσειράς ή η γραμματοσειρά δεν είναι εγκατεστημένη τοπικά, μπορεί να εμφανίζονται ως μπλοκ σε pdf, εικόνα ή XPS. Ορίστε την Προεπιλεγμένη Γραμματοσειρά όπως MingLiu ή MS Gothic για να εμφανίζονται χαρακτήρες.

```csharp
public virtual string DefaultFont { get; set; }
```
