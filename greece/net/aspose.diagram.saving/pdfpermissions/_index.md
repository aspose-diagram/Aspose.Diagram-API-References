## PdfPermissions enumeration

Καθορίζει δικαιώματα χρήστη για έγγραφο PDF.

```csharp
[Flags]
public enum PdfPermissions
```

### Αξίες

| Ονομα | αξία | Περιγραφή |
| --- | --- | --- |
| DisallowAll | `0` | Απαγορεύει όλες τις λειτουργίες στο έγγραφο PDF. Αυτή είναι η προεπιλεγμένη τιμή. |
| Printing | `4` | Επιτρέπει την εκτύπωση του εγγράφου. |
| ModifyContents | `8` | Επιτρέπει την τροποποίηση των περιεχομένων του εγγράφου. |
| ContentCopy | `10` | Επιτρέπει την αντιγραφή ή με άλλο τρόπο εξαγωγή κειμένου και γραφικών από το έγγραφο, συμπεριλαμβανομένης της εξαγωγής για λόγους προσβασιμότητας. |
| ModifyAnnotations | `20` | Επιτρέπει την προσθήκη ή την τροποποίηση σχολιασμών κειμένου. Όταν χρησιμοποιείτε κρυπτογράφηση 40-bit RC4, αυτή η επιλογή επιτρέπει επίσης τη συμπλήρωση πεδίων φόρμας. |
| FillIn | `100` | Επιτρέπει τη συμπλήρωση εντύπων και την υπογραφή του εγγράφου. Όταν χρησιμοποιείτε κρυπτογράφηση 40-bit RC4, αυτή η επιλογή αγνοείται και επιτρέπεται η συμπλήρωση της φόρμας κάθε φορά που ορίζεται το ModifyAnnotations. |
| ContentCopyForAccessibility | `200` | Επιτρέπει την εξαγωγή κειμένου και γραφικών για υποστήριξη της προσβασιμότητας σε χρήστες με ειδικές ανάγκες ή για άλλους σκοπούς. Όταν χρησιμοποιείτε κρυπτογράφηση 40-bit RC4, αυτή η επιλογή αγνοείται και επιτρέπεται η προσβασιμότητα κάθε φορά που ορίζεται το ContentCopy. |
| DocumentAssembly | `400` | Επιτρέπει τη συναρμολόγηση του εγγράφου: εισαγωγή, περιστροφή ή διαγραφή σελίδων και δημιουργία στοιχείων πλοήγησης, όπως σελιδοδείκτες ή μικρογραφίες. Όταν χρησιμοποιείτε κρυπτογράφηση 40-bit RC4, αυτή η επιλογή αγνοείται και επιτρέπεται η συναρμολόγηση εγγράφων όταν έχει οριστεί το ModifyContents. |
| HighResolutionPrinting | `804` | Επιτρέπει την εκτύπωση του εγγράφου στην υψηλότερη δυνατή ανάλυση. Όταν χρησιμοποιείτε κρυπτογράφηση 40-bit RC4, αυτή η επιλογή αγνοείται και επιτρέπεται η εκτύπωση υψηλής ανάλυσης όταν έχει οριστεί η εκτύπωση. |
| AllowAll | `FFFF` | Επιτρέπει όλες τις λειτουργίες στο έγγραφο PDF. |
