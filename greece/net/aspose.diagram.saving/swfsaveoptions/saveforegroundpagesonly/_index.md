## SWFSaveOptions.SaveForegroundPagesOnly property

Καθορίζει εάν όλες οι σελίδες θα αποθηκεύονται σε εικόνα ή μόνο σε πρώτο πλάνο.

```csharp
public bool SaveForegroundPagesOnly { get; set; }
```

### Παρατηρήσεις

Εάν είναι true - αποδίδονται μόνο σελίδες στο προσκήνιο (με φόντο εάν υπάρχουν). Εάν είναι false - αποδίδονται σελίδες προσκηνίου (με φόντο εάν υπάρχουν) μετά από αυτές τις κενές σελίδες παρασκηνίου. Μπορεί να επιστρέψει true μόνο όταν PageCount &gt; 1. Η προεπιλεγμένη τιμή είναι false .
