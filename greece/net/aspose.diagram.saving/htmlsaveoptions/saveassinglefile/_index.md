## HTMLSaveOptions.SaveAsSingleFile property

Υποδεικνύει εάν αποθηκεύεται η html ως μεμονωμένο αρχείο. Η προεπιλεγμένη τιμή είναι false.

```csharp
public bool SaveAsSingleFile { get; set; }
```

### Παρατηρήσεις

Εάν υπάρχουν πολλές σελίδες, αυτές οι σελίδες και άλλοι πόροι πρέπει να αποθηκευτούν σε ξεχωριστά αρχεία. Για ορισμένα σενάρια, ο χρήστης ίσως χρειαστεί να λάβει μόνο ένα αρχείο που προκύπτει, όπως για την ευκολία της μεταφοράς. Εάν ναι, ο χρήστης μπορεί να ορίσει αυτήν την ιδιότητα ως true.
