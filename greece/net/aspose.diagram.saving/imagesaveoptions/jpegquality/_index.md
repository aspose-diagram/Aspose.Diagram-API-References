## ImageSaveOptions.JpegQuality property

Λαμβάνει ή ορίζει μια τιμή που καθορίζει την ποιότητα των δημιουργούμενων εικόνων JPEG.

```csharp
public int JpegQuality { get; set; }
```

### Παρατηρήσεις

Έχει αποτέλεσμα μόνο κατά την αποθήκευση σε JPEG. Χρησιμοποιήστε αυτήν την ιδιότητα για να λάβετε ή να ορίσετε την ποιότητα των εικόνων που δημιουργούνται κατά την αποθήκευση σε μορφή JPEG. Η τιμή μπορεί να ποικίλλει από 0 έως 100 όπου 0 σημαίνει χειρότερη ποιότητα, μέγιστη συμπίεση και 100 σημαίνει καλύτερη ποιότητα αλλά ελάχιστη συμπίεση. Η προεπιλεγμένη τιμή είναι 95.
