## PrintSaveOptions.SaveForegroundPagesOnly property

Καθορίζει εάν όλες οι σελίδες θα εκτυπωθούν ή μόνο στο προσκήνιο.

```csharp
public bool SaveForegroundPagesOnly { get; set; }
```

### Παρατηρήσεις

Αν true - εκτυπώνονται μόνο σελίδες προσκηνίου (με φόντο εάν υπάρχουν). Εάν είναι false - εκτυπώνονται σελίδες προσκηνίου (με φόντο εάν υπάρχουν) μετά από αυτές τις κενές σελίδες φόντου. Μπορεί να επιστρέψει true μόνο όταν PageCount &gt; 1. Η προεπιλεγμένη τιμή είναι false .
