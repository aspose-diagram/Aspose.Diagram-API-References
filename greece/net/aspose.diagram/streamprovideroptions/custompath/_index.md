## StreamProviderOptions.CustomPath property

Η προσαρμοσμένη διαδρομή χρήστη (URL) που έχει αποθηκευτεί στο αρχείο html που δημιουργήθηκε για την αναφερόμενη πηγή. Εάν δεν ορίζεται από τον χρήστη, θα χρησιμοποιηθεί το DefaultPath. Για παράδειγμα, τα δεδομένα φύλλου θα αποθηκευτούν από τον χρήστη στο d:/sheet001.htm, το url που χρησιμοποιείται στο κύριο αρχείο html θα πρέπει να είναι "d:/sheet001.htm" ή άλλη έγκυρη σχετική διαδρομή στην οποία μπορεί να προσπελαστεί το κύριο αρχείο html.

```csharp
public string CustomPath { set; }
```
