## GluedShapesFlags enumeration

Καθορίζει σταθερές που προσδιορίζουν ποια σχήματα θα επιστρέψουν, με βάση τη διάσταση και την κατευθυντικότητα των σημείων σύνδεσης που είναι κολλημένα σε ένα συγκεκριμένο σχήμα. πέρασε στη μέθοδο Shape.GluedShapes.

```csharp
public enum GluedShapesFlags
```

### Αξίες

| Ονομα | αξία | Περιγραφή |
| --- | --- | --- |
| GluedShapesAll1D | `0` | Επιστροφή αναγνωριστικών όλων των σχημάτων 1-Δ που είναι κολλημένα σε αυτό το σχήμα. |
| GluedShapesIncoming1D | `1` | Επιστροφή αναγνωριστικών σχημάτων 1-Δ των οποίων τα τελικά σημεία είναι κολλημένα σε αυτό το σχήμα. |
| GluedShapesOutgoing1D | `2` | Επιστροφή αναγνωριστικών σχημάτων 1-Δ των οποίων τα σημεία έναρξης είναι κολλημένα σε αυτό το σχήμα. |
| GluedShapesAll2D | `3` | Επιστρέψτε όλα τα σχήματα 2-Δ που είναι κολλημένα σε αυτό το σχήμα και όλα τα σχήματα 2-Δ στα οποία είναι κολλημένο αυτό το σχήμα. |
| GluedShapesIncoming2D | `4` | Εάν το αντικείμενο προέλευσης είναι 1-D σχήμα, επιστρέψτε αναγνωριστικά με σχήμα 2-D στα οποία είναι κολλημένο το σημείο έναρξης. Εάν το αντικείμενο προέλευσης είναι σχήμα 2-Δ, επιστρέψτε αναγνωριστικά σχημάτων 2-Δ που είναι κολλημένα σε αυτό το σχήμα. |
| GluedShapesOutgoing2D | `5` | Εάν το αντικείμενο προέλευσης είναι 1-D σχήμα, επιστρέψτε αναγνωριστικά με σχήμα 2-D στα οποία είναι κολλημένο το τελικό σημείο. Εάν το αντικείμενο προέλευσης είναι σχήμα 2-Δ, επιστρέψτε αναγνωριστικά σχημάτων 2-Δ στα οποία είναι κολλημένο αυτό το σχήμα. |
