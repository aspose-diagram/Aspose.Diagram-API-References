## InfiniteLine class

Περιέχει στοιχεία που καθορίζουν τις συντεταγμένες x και y δύο σημείων σε μια άπειρη ευθεία. Τα στοιχεία Χ και Υ καθορίζουν τις συντεταγμένες x και y του πρώτου σημείου και τα στοιχεία Α και Β προσδιορίζουν τις συντεταγμένες x και y του δεύτερου σημείου.

```csharp
public class InfiniteLine : Coordinate
```
