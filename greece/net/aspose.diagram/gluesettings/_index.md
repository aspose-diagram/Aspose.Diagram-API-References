## GlueSettings enumeration

Οι τιμές bit υποδεικνύουν ότι μια συγκεκριμένη ρύθμιση κόλλας είναι ενεργοποιημένη ή απενεργοποιημένη. Η τιμή μπορεί να είναι ένα άθροισμα των τιμών:

```csharp
[Flags]
public enum GlueSettings
```

### Αξίες

| Ονομα | αξία | Περιγραφή |
| --- | --- | --- |
| None | `0` | Η κόλλα είναι ενεργοποιημένη, αλλά δεν είναι ενεργοποιημένες άλλες ρυθμίσεις κόλλας. |
| Guides | `1` | Κόλλα στους οδηγούς. |
| Handles | `2` | Κόλλα στις λαβές. |
| Vertices | `4` | Κόλλα στις κορυφές. |
| ConnectionPoints | `8` | Κόλλα στα σημεία σύνδεσης. |
| Geometry | `20` | Κόλλα στη γεωμετρία. |
| Disabled | `8000` | Η κόλλα είναι απενεργοποιημένη |
| Undefined | `FFFFFFFF80000000` | Undefined. |
