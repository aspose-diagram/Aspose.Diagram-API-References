## Master class

Περιέχει στοιχεία που ορίζουν έναν κύριο για το έγγραφο. Ο κύριος είναι ένα σχήμα σε ένα στένσιλ που χρησιμοποιείτε επανειλημμένα για να δημιουργήσετε σχέδια. Όταν σύρετε ένα σχήμα από ένα στένσιλ στη σελίδα σχεδίασης, το σχήμα γίνεται μια εμφάνιση αυτού του κύριου και ένα τοπικό αντίγραφο του κύριου περιλαμβάνεται στο έγγραφο.

```csharp
public class Master : IDisposable
```
