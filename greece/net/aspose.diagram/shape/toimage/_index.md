## ToImage(string, ImageSaveOptions) {#toimage_1}

Δημιουργεί την εικόνα σχήματος και την αποθηκεύει σε ένα αρχείο. Η επέκταση του ονόματος αρχείου καθορίζει τη μορφή της εικόνας.

```csharp
public void ToImage(string imageFile, ImageSaveOptions options)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| imageFile | String | Το όνομα αρχείου εικόνας με πλήρη διαδρομή. |
| options | ImageSaveOptions | Πρόσθετες επιλογές δημιουργίας εικόνας |

### Παρατηρήσεις

Η μορφή της εικόνας καθορίζεται χρησιμοποιώντας την επέκταση του ονόματος αρχείου. Για παράδειγμα, εάν καθορίσετε "myfile.png", τότε η εικόνα θα αποθηκεύεται σε μορφή PNG. Αναγνωρίζονται οι ακόλουθες επεκτάσεις αρχείων: .bmp, .gif, .png, .jpg, .jpeg, .tiff, .tif, .emf.
