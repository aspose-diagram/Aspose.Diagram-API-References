## NURBSTo class

Περιέχει τις συντεταγμένες x και y, τη θέση του δεύτερου έως του τελευταίου κόμβου, τη θέση του τελευταίου βάρους, τη θέση του πρώτου κόμβου, τη θέση του πρώτου βάρους και τον τύπο για μια ανομοιόμορφη ορθολογική B-spline (NURBS). Αυτές οι πληροφορίες καθορίζονται στα στοιχεία X, Y, A, B, C, D και E, αντίστοιχα.

public class NURBSTo : Coordinate
