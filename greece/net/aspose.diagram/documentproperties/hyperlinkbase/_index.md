## DocumentProperties.HyperlinkBase property

Περιέχει τη διαδρομή που θα χρησιμοποιηθεί για σχετικούς υπερσυνδέσμους (υπερσυνδέσεις για τις οποίες η θέση του συνδεδεμένου αρχείου περιγράφεται σε σχέση με το διάγραμμα του Microsoft Visio). Από προεπιλογή, μια διαδρομή υπερ-σύνδεσης είναι σχετική με το τρέχον έγγραφο, εκτός εάν ορίζεται διαφορετική διαδρομή σε αυτό το στοιχείο. Το μέγιστο μήκος είναι 256 χαρακτήρες.

```csharp
public string HyperlinkBase { get; set; }
```
