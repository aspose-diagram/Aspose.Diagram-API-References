## PageLayout.LineJumpFactorY property

Καθορίζει το μέγεθος των μεταπηδήσεων γραμμής σε κάθετα τμήματα δυναμικών συνδέσεων στη σελίδα, ως ποσοστό της τιμής του στοιχείου LineToLineY (το οποίο καθορίζει την κατακόρυφη απόσταση μεταξύ όλων των υποδοχών σύνδεσης στη σελίδα σχεδίασης). Αυτό το στοιχείο μπορεί να περιέχει μια τιμή από 0 έως 10.

public DoubleValue LineJumpFactorY { get; }
