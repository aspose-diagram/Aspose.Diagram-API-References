## EllipticalArcTo.D property

Ο λόγος του κύριου άξονα ενός τόξου προς τον δευτερεύοντα άξονά του. Παρά τη συνήθη σημασία αυτών των λέξεων, ο κύριος άξονας δεν χρειάζεται να είναι μεγαλύτερος από τον δευτερεύοντα άξονα, επομένως αυτός ο λόγος δεν χρειάζεται να είναι μεγαλύτερος από 1. Ορίζοντας αυτό το στοιχείο σε τιμή μικρότερη ή ίση με 0 ή μεγαλύτερη από 1000 μπορεί να οδηγήσει σε απρόβλεπτα αποτελέσματα.

public DoubleValue D { get; set; }
