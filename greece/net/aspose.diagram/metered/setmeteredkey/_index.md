## Metered.SetMeteredKey method

Ορίζει το μετρημένο δημόσιο και ιδιωτικό κλειδί. Εάν αγοράσετε μετρημένη άδεια, κατά την έναρξη της εφαρμογής, αυτό το API θα πρέπει να καλείται, κανονικά, αυτό είναι αρκετό. Ωστόσο, εάν αποτυγχάνετε πάντα να ανεβάζετε δεδομένα κατανάλωσης και υπερβείτε τις 24 ώρες, η άδεια χρήσης θα ρυθμιστεί σε κατάσταση αξιολόγησης. Για να αποφύγετε μια τέτοια περίπτωση, θα πρέπει να ελέγχετε τακτικά την κατάσταση της άδειας, εάν είναι κατάσταση αξιολόγησης, καλέστε ξανά αυτό το API.

```csharp
public void SetMeteredKey(string publicKey, string privateKey)
```
