## AutoLinkComparison.ContextTypeLabel property

Εάν η τιμή ContextType είναι 2 ή 3, αυτό το χαρακτηριστικό απαιτείται για να οριστεί μια σύγκριση. Για ContextType = 2, το ContextTypeLabel πρέπει να είναι η ετικέτα στοιχείου δεδομένων σχήματος και εάν ContextType = 3, το ContextTypeLabel πρέπει να είναι το όνομα της τοπικής γραμμής.

public string ContextTypeLabel { get; set; }
