## EventItem class

Ενθυλακώνει έναν κωδικό συμβάντος. Ένα στοιχείο EventItem μπορεί να ενεργοποιήσει δύο είδη ενεργειών: μπορεί να εκτελέσει ένα πρόσθετο ή μπορεί να στείλει μια ειδοποίηση για το συμβάν στο πρόγραμμα κλήσης.
