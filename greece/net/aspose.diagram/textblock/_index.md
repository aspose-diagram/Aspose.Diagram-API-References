## TextBlock class

Περιέχει στοιχεία που καθορίζουν τη στοίχιση, τα περιθώρια και τις προεπιλεγμένες θέσεις του κειμένου στο μπλοκ κειμένου ενός σχήματος.

```csharp
public class TextBlock : ICloneable
```
