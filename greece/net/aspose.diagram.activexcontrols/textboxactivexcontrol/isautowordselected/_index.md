## TextBoxActiveXControl.IsAutoWordSelected property

Καθορίζει τη βασική μονάδα που χρησιμοποιείται για την επέκταση μιας επιλογής. Το True προσδιορίζει ότι η βασική μονάδα είναι ένας μεμονωμένος χαρακτήρας. false καθορίζει ότι η βασική μονάδα είναι μια ολόκληρη λέξη.
