## TextBoxActiveXControl.EnterKeyBehavior property

Καθορίζει τη συμπεριφορά του πλήκτρου ENTER. True καθορίζει ότι πατώντας το ENTER θα δημιουργηθεί μια νέα γραμμή. False καθορίζει ότι πατώντας ENTER θα μετακινηθεί η εστίαση στο επόμενο αντικείμενο με τη σειρά της καρτέλας.

public bool EnterKeyBehavior { get; set; }
