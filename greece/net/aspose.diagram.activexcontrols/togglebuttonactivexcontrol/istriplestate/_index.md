## ToggleButtonActiveXControl.IsTripleState property

Υποδεικνύει πώς το καθορισμένο στοιχείο ελέγχου θα εμφανίζει τιμές Null.

```csharp
public bool IsTripleState { get; set; }
```

### Παρατηρήσεις

**Σύνθεση**

**Περιγραφή**

Αληθής

Το στοιχείο ελέγχου θα εναλλάσσεται μεταξύ των καταστάσεων για τις τιμές Yes, No και Null. Το στοιχείο ελέγχου εμφανίζεται θαμπό (γκρι) όταν η ιδιότητα Value έχει οριστεί σε Null.

Ψευδής

(Προεπιλογή) Το στοιχείο ελέγχου θα εναλλάσσεται μεταξύ των καταστάσεων για τις τιμές Ναι και Όχι. Οι μηδενικές τιμές εμφανίζονται σαν να ήταν χωρίς τιμές.
