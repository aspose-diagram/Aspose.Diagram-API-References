### getContextTypeLabel() {#getContextTypeLabel--}
```
public String getContextTypeLabel()
```


Εάν η τιμή ContextType είναι 2 ή 3, αυτό το χαρακτηριστικό απαιτείται για τον ορισμό μιας σύγκρισης. Για ContextType = 2, το ContextTypeLabel πρέπει να είναι η ετικέτα του στοιχείου δεδομένων σχήματος, και εάν ContextType = 3, το ContextTypeLabel πρέπει να είναι το τοπικό όνομα γραμμής.

**Returns:**
