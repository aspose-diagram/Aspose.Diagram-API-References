### AUTOMATICALLY {#AUTOMATICALLY}

```
public static final int AUTOMATICALLY
```

### DEPENDS_ON_DRAWING_SIZE_TYPE {#DEPENDS-ON-DRAWING-SIZE-TYPE}
```
public static final int DEPENDS_ON_DRAWING_SIZE_TYPE
```


Το αν η σελίδα αλλάζει μέγεθος αυτόματα εξαρτάται από την τιμή του κελιού DrawingSizeType. Εάν DrawingSizeType = 0 (το μέγεθος της σελίδας σχεδίου είναι το ίδιο με το μέγεθος της εκτυπωμένης σελίδας), η σελίδα αλλάζει μέγεθος αυτόματα. Εάν η τιμή του DrawingSizeType είναι οποιαδήποτε άλλη από το μηδέν (0), η σελίδα δεν αλλάζει μέγεθος αυτόματα.
