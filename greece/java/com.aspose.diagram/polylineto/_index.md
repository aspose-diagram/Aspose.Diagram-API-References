Περιέχει τις συντεταγμένες x και y του τελευταίου σημείου μιας πολυγραμμής και έναν τύπο πολυγραμμής. Οι συντεταγμένες καθορίζονται στα στοιχεία X και Y, και ο τύπος καθορίζεται στο στοιχείο A.
