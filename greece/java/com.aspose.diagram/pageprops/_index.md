### getDrawingScale() {#getDrawingScale--}
```
public DoubleValue getDrawingScale()
```


Αναπαριστά την τιμή της μονάδας σχεδίασης στην τρέχουσα κλίμακα σχεδίασης. Η κλίμακα σχεδίασης για τη σελίδα είναι ο λόγος της μονάδας σελίδας που περιέχεται στο στοιχείο PageScale προς τη μονάδα σχεδίασης. Οι μονάδες αυτού του στοιχείου καθορίζουν τις προεπιλεγμένες μονάδες μήκους (DL) για τη σελίδα.
