Καθορίζει τη μορφή του κειμένου ενός σχήματος. Όλα τα κεφαλαία (uppercase) γράμματα (1) και τα αρχικά κεφαλαία (2) δεν αλλάζουν την εμφάνιση του κειμένου που εισήχθη με όλα τα κεφαλαία. Το κείμενο πρέπει να εισαχθεί με πεζά γράμματα για να εμφανιστούν αυτές οι επιλογές.
