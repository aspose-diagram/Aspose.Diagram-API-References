### getD() {#getD--}
```
public DoubleValue getD()
```


Ο λόγος του κύριου άξονα μιας καμπύλης προς τον δευτερεύοντα άξονα. Παρά τη συνήθη σημασία αυτών των λέξεων, ο κύριος άξονας δεν χρειάζεται να είναι μεγαλύτερος από τον δευτερεύοντα, επομένως αυτός ο λόγος δεν χρειάζεται να είναι μεγαλύτερος από 1. Ο ορισμός αυτού του στοιχείου σε τιμή μικρότερη ή ίση με 0 ή μεγαλύτερη από 1000 μπορεί να οδηγήσει σε απρόβλεπτα αποτελέσματα.
