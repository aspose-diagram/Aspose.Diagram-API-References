### NO_ADV_CONFIG {#NO-ADV-CONFIG}
```
public static final int NO_ADV_CONFIG
```


Περιορίζει τον έλεγχο που έχουν οι χρήστες σχετικά με το πώς ανανεώνεται το σύνολο δεδομένων στο παράθυρο διαλόγου Διαμόρφωση Ανανέωσης για το σύνολο δεδομένων. Συγκεκριμένα, οι χρήστες δεν μπορούν να αλλάξουν το πρωτεύον κλειδί ή να καθορίσουν πότε θα αντικατασταθούν τα δεδομένα σχήματος· ωστόσο, μπορούν να ορίσουν το διάστημα ανανέωσης και να αλλάξουν την πηγή δεδομένων.
