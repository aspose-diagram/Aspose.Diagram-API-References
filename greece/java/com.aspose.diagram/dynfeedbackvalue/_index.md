Καθορίζει τον τύπο της οπτικής ανάδρασης που παρέχεται στους χρήστες όταν σύρουν έναν σύνδεσμο. Όταν απελευθερωθεί το κουμπί του ποντικιού, το τελικό σχήμα του συνδέσμου δεν επηρεάζεται από αυτή τη ρύθμιση. Αυτό το στοιχείο δεν εφαρμόζεται σε δρομολογήσιμους συνδέσμους.
