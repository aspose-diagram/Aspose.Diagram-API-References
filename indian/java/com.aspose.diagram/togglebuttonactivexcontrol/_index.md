| ------- | ----------------------------------------------------------------------------------------------------------------------------------------------- |
| सेटिंग | विवरण |
| True    | नियंत्रण Yes, No, और Null मानों के लिए अवस्थाओं के बीच चक्र करेगा। जब इसका Value गुण Null पर सेट होता है तो नियंत्रण धुंधला (ग्रे) दिखाई देता है। |
| False   | (Default) नियंत्रण Yes और No मानों के लिए अवस्थाओं के बीच चक्र करेगा। Null मानों को No मानों की तरह प्रदर्शित किया जाता है। |

|
