### getContextTypeLabel() {#getContextTypeLabel--}
```
public String getContextTypeLabel()
```


यदि ContextType मान 2 या 3 है, तो तुलना परिभाषित करने के लिए यह विशेषता आवश्यक है। ContextType = 2 के लिए, ContextTypeLabel को शेप डेटा आइटम लेबल होना चाहिए, और यदि ContextType = 3 है, तो ContextTypeLabel को स्थानीय पंक्ति नाम होना चाहिए।
