एक अनुक्रमित मान जो निम्नलिखित में से एक हो सकता है: Drawing, Sheet, Stencil, या Icon। WindowType='Stencil' वाला एक Window तत्व अपने पैरेंट ड्राइंग विंडो (WindowType='Drawing') के बाद और किसी भी अन्य ड्राइंग विंडो तत्वों से पहले प्रकट होना चाहिए।
