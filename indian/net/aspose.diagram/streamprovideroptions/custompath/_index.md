## StreamProviderOptions.CustomPath property

संदर्भित स्रोत के लिए जनरेट की गई html फ़ाइल में सहेजा गया उपयोगकर्ता कस्टम पथ (URL)। यदि उपयोगकर्ता द्वारा परिभाषित नहीं किया गया है, तो DefaultPath का उपयोग किया जाएगा। उदाहरण के लिए, शीट डेटा उपयोगकर्ता द्वारा d:/sheet001.htm में सहेजा जाएगा, मुख्य html फ़ाइल में उपयोग किया जाने वाला url "d:/sheet001.htm" होना चाहिए या अन्य मान्य सापेक्ष पथ जिसे मुख्य html फ़ाइल द्वारा एक्सेस किया जा सकता है.

public string CustomPath { set; }
