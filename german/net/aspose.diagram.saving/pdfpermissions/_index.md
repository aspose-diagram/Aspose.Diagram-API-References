## PdfPermissions enumeration

Gibt Benutzerberechtigungen für das PDF-Dokument an.

```csharp
[Flags]
public enum PdfPermissions
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| DisallowAll | `0` | Verbietet alle Operationen auf dem PDF-Dokument. Dies ist der Standardwert. |
| Printing | `4` | Ermöglicht das Drucken des Dokuments. |
| ModifyContents | `8` | Ermöglicht das Ändern des Inhalts des Dokuments. |
| ContentCopy | `10` | Ermöglicht das Kopieren oder anderweitige Extrahieren von Text und Grafiken aus dem Dokument, einschließlich Extrahieren für Zugänglichkeitszwecke. |
| ModifyAnnotations | `20` | Ermöglicht das Hinzufügen oder Ändern von Textanmerkungen. Bei Verwendung der RC4 40-Bit-Verschlüsselung ermöglicht diese Option auch das Ausfüllen von Formularfeldern. |
| FillIn | `100` | Ermöglicht das Ausfüllen von Formularen und das Unterschreiben des Dokuments. Bei Verwendung der RC4-40-Bit-Verschlüsselung wird diese Option ignoriert und das Ausfüllen des Formulars ist immer dann zulässig, wenn ModifyAnnotations festgelegt ist. |
| ContentCopyForAccessibility | `200` | Ermöglicht das Extrahieren von Text und Grafiken zur Unterstützung der Zugänglichkeit für behinderte Benutzer oder für andere Zwecke. Bei Verwendung der RC4-40-Bit-Verschlüsselung wird diese Option ignoriert und der Zugriff wird immer dann zugelassen, wenn ContentCopy festgelegt ist. |
| DocumentAssembly | `400` | Ermöglicht das Zusammenstellen des Dokuments: Einfügen, Drehen oder Löschen von Seiten und Erstellen von Navigationselementen wie Lesezeichen oder Miniaturbildern. Bei Verwendung von RC4 40-Bit-Verschlüsselung wird diese Option ignoriert und die Dokumentzusammenstellung wird zugelassen, wenn ModifyContents festgelegt ist. |
| HighResolutionPrinting | `804` | Ermöglicht das Drucken des Dokuments mit der höchstmöglichen Auflösung. Bei Verwendung der RC4 40-Bit-Verschlüsselung wird diese Option ignoriert und das Drucken mit hoher Auflösung wird zugelassen, wenn Drucken eingestellt ist. |
| AllowAll | `FFFF` | Lässt alle Operationen auf dem PDF-Dokument zu. |
