## RenderingSaveOptions.EmfRenderSetting property

Einstellung zum Rendern der EMF-Metadatei.

```csharp
public EmfRenderSetting EmfRenderSetting { get; set; }
```

### Bemerkungen

Als „EMF+ Dual“ gekennzeichnete EMF-Metadateien können sowohl EMF+-Datensätze als auch EMF-Datensätze enthalten. Beide Arten von Datensätzen können zum Rendern des Bildes verwendet werden, nur EMF+-Datensätze oder nur EMF-Datensätze. Wann!:EmfRenderSetting.EmfPlusPrefer gesetzt ist, dann werden EMF+-Datensätze geparst, andernfalls werden nur EMF-Datensätze geparst. Der Standardwert ist!:EmfRenderSetting.EmfOnly.
