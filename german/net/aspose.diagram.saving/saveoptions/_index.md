### Bemerkungen

Eine Instanz der SaveOptions-Klasse oder einer beliebigen abgeleiteten Klasse wird an den Stream Save oder die Zeichenfolge Save überladen übergeben, damit der Benutzer beim Speichern eines Dokuments benutzerdefinierte Optionen definieren kann.
