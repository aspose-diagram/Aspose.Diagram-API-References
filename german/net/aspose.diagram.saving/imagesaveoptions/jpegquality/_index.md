## ImageSaveOptions.JpegQuality property

Ruft einen Wert ab oder legt einen Wert fest, der die Qualität der generierten JPEG-Bilder bestimmt.

```csharp
public int JpegQuality { get; set; }
```

### Bemerkungen

Hat nur Auswirkung beim Speichern im JPEG-Format. Verwenden Sie diese Eigenschaft, um die Qualität der generierten Bilder beim Speichern im JPEG-Format abzurufen oder festzulegen. Der Wert kann zwischen 0 und 100 variieren, wobei 0 die schlechteste Qualität, aber maximale Komprimierung und 100 die beste bedeutet Qualität, aber minimale Komprimierung. Der Standardwert ist 95.
