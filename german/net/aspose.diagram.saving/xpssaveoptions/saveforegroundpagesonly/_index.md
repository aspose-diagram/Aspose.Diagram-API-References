## XPSSaveOptions.SaveForegroundPagesOnly property

Gibt an, ob alle Seiten im Bild oder nur im Vordergrund gespeichert werden.

```csharp
public bool SaveForegroundPagesOnly { get; set; }
```

### Bemerkungen

Wenn wahr – nur Vordergrundseiten gerendert (mit Hintergrund, falls vorhanden). Wenn falsch – Vordergrundseiten gerendert (mit Hintergrund, falls vorhanden), danach leere Hintergrundseiten. Kann nur wahr zurückgeben, wenn PageCount &gt; 1. Der Standardwert ist falsch .
