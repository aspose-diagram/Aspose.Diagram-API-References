## TextBoxActiveXControl.EnterFieldBehavior property

Gibt das Auswahlverhalten beim Aufrufen des Steuerelements an. True gibt an, dass die Auswahl seit dem letzten Mal, als das Steuerelement aktiv war, unverändert bleibt. False gibt an, dass der gesamte Text im Steuerelement ausgewählt wird, wenn das Steuerelement eingegeben wird.
