## CheckBoxActiveXControl.IsTripleState property

Gibt an, wie das angegebene Steuerelement Nullwerte anzeigt.


### Bemerkungen

**Einstellung**

**Beschreibung**

WAHR

Die Steuerung durchläuft die Zustände für Ja-, Nein- und Nullwerte. Das Steuerelement wird abgeblendet (grau) angezeigt, wenn seine Value-Eigenschaft auf Null festgelegt ist.

FALSCH

(Standard) Die Steuerung durchläuft die Zustände für Ja- und Nein-Werte. Nullwerte werden so angezeigt, als wären sie keine Werte.
