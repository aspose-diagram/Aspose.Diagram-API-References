## Prop.Prompt property

Eingabeaufforderungselement gibt beschreibenden oder anweisenden Text an, der Benutzern im Dialogfeld „Benutzerdefinierte Eigenschaften“ angezeigt wird, wenn die Eigenschaft ausgewählt wird. Dieser Text wird auch als QuickInfo angezeigt, wenn der Mauszeiger über der Eigenschaft im Fenster „Benutzerdefinierte Eigenschaften“ gehalten wird.
