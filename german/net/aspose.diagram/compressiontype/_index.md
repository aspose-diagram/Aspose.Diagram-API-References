## CompressionType enumeration

Dieses Attribut ist nur sinnvoll, wenn es sich bei den Fremddaten um ein rasterbasiertes Fremdobjekt handelt, z. B. eine DIB-, JPG-, PNG-, TIFF- oder GIF-Datei. Der Wert gibt den auf die Datei angewendeten Komprimierungstyp an.

| Name | Wert | Beschreibung |

| JPEG | `1` | JPEG-Komprimierung. |
| GIF | `2` | GIF-Komprimierung. |
| TIFF | `3` | TIFF-Komprimierung. |
| PNG | `4` | PNG-Komprimierung. |
