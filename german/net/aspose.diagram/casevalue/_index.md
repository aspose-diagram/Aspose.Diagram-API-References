## CaseValue enumeration

Legt die Groß-/Kleinschreibung des Textes einer Form fest. Alle Großbuchstaben (1) und Anfangsbuchstaben (2) ändern nicht das Erscheinungsbild von Text, der ausschließlich in Großbuchstaben eingegeben wurde. Der Text muss in Kleinbuchstaben eingegeben werden, damit diese Optionen wirken.

```csharp
public enum CaseValue
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| NormalCase | `0` | Normalfall. |
| AllCapitalLetters | `1` | Alle Großbuchstaben (Großbuchstaben). |
| InitialCapitalLettersOnly | `2` | Nur Anfangsbuchstaben. |
| Undefined | `-2147483648` | Undefiniert. |
