## Master.MatchByName property

Das MatchByName-Attribut bestimmt, wie Microsoft Visio entscheidet, ob ein Dokumentmaster bereits vorhanden ist, wenn eine Instanz eines Masters auf dem Zeichenblatt abgelegt wird. Es ermöglicht, dass an einem Dokumentenmaster vorgenommene Änderungen auf neue Instanzen des Masters angewendet werden, selbst wenn die Instanzen aus einer eigenständigen Schablonendatei gezogen werden.
