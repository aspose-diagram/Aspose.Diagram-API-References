## RuleValue.Formula property

Repräsentiert die Formel des Elements. Dieses Attribut kann einen der folgenden Strings enthalten: „someFormula“, wenn die Formel lokal existiert, „No Formula“, wenn die Formel lokal gelöscht oder gesperrt ist, oder „Inh“, wenn die Formel vererbt wird. Wenn das Attribut nicht vorhanden ist, ist die Formel des Elements eine einfache Konstante.

public string Formula { get; set; }
