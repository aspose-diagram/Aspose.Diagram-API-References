## ToImage(string, ImageSaveOptions) {#toimage_1}

Erstellt das Formbild und speichert es in einer Datei. Die Erweiterung des Dateinamens bestimmt das Format des Bildes.

```csharp
public void ToImage(string imageFile, ImageSaveOptions options)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| imageFile | String | Der Bilddateiname mit vollständigem Pfad. |
| options | ImageSaveOptions | Zusätzliche Bilderzeugungsoptionen |

### Bemerkungen

Das Format des Bildes wird durch Verwendung der Erweiterung des Dateinamens angegeben. Wenn Sie beispielsweise "myfile.png" angeben, wird das Bild im PNG-Format gespeichert. Folgende Dateierweiterungen werden erkannt: .bmp, .gif, .png, .jpg, .jpeg, .tiff, .tif, .emf.
