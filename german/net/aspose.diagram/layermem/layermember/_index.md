## LayerMem.LayerMember property

Gibt die Ebene oder Ebenen an, denen die Form zugewiesen ist. Die Ebenenzuweisung wird basierend auf dem nullbasierten Index der Ebenen für die Seite angegeben. Wenn eine Form mehr als einer Ebene zugewiesen ist, wird jeder Ebenenindex durch ein Semikolon getrennt angezeigt.
