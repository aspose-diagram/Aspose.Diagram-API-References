## DocumentProperties.HyperlinkBase property

Enthält den Pfad, der für relative Hyperlinks verwendet werden soll (Hyperlinks, für die der Speicherort der verknüpften Datei in Bezug auf das Microsoft Visio-Diagramm beschrieben wird). Standardmäßig ist ein Hyperlink-Pfad relativ zum aktuellen Dokument, es sei denn, in diesem Element wird ein anderer Pfad angegeben. Die maximale Länge beträgt 256 Zeichen.
