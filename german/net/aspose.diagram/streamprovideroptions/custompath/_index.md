## StreamProviderOptions.CustomPath property

Der benutzerdefinierte Pfad (URL) des Benutzers, der in der generierten HTML-Datei für die angegebene Quelle gespeichert ist. Wenn nicht vom Benutzer definiert, wird DefaultPath verwendet. Beispielsweise werden die Blattdaten vom Benutzer in d:/sheet001.htm gespeichert, die in der Haupt-HTML-Datei verwendete URL sollte "d:/sheet001.htm" sein anderer gültiger relativer Pfad, auf den von der Haupt-HTML-Datei zugegriffen werden kann.

```csharp
public string CustomPath { set; }
```
