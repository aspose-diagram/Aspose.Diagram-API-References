## DisplayModeValue enumeration

Wenn es in einem Group-Element enthalten ist, gibt das DisplayMode-Element an, wie eine Gruppenform und ihre Mitglieder angezeigt werden. Wenn es in einem SmartTagDef-Element enthalten ist, bestimmt das DisplayMode-Element, ob das Smarttag angezeigt wird, wenn der Benutzer die Maus darüber hält das Tag, wenn die Form ausgewählt ist, oder die ganze Zeit.

```csharp
public enum DisplayModeValue
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| HidesShapeText | `0` | Blendet die Gruppenform und den Text aus. |
| DisplaysShapeBehindMemberShapes | `1` | Zeigt das Gruppen-Shape hinter Mitglieds-Shapes an. |
| DisplaysShapeFrontMemberShapes | `2` | Zeigt das Gruppen-Shape vor den Mitglieds-Shapes an. |
| Undefined | `-2147483648` | Undefiniert. |
