## Window.StencilGroup property

Gibt die Gruppe der zusammengeführten Schablonenfenster an, zu denen das Fenster gehört. Dieses Attribut ist nur für Window-Elemente relevant, deren WindowType-Attribut Stencil ist, und nur dann, wenn das Schablonenfenster Teil einer zusammengeführten Gruppe von Schablonenfenstern ist. Alle Schablonenfenster, die Teil derselben zusammengeführten Gruppe sind, haben denselben StencilGroup-Elementwert.

public string StencilGroup { get; set; }
