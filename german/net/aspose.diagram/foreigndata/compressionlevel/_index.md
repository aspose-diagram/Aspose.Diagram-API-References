## ForeignData.CompressionLevel property

Dieses Attribut ist nur sinnvoll, wenn es sich bei den Fremddaten um ein rasterbasiertes Fremdobjekt handelt, z. B. eine DIB-, JPG-, PNG-, TIFF- oder GIF-Datei. Der Wert gibt die Komprimierungsstufe an, die auf die Datei angewendet wird. Die Komprimierungsstufe wird in Hundertstelprozent gemessen.

public double CompressionLevel { get; set; }
