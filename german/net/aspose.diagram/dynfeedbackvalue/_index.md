## DynFeedbackValue enumeration

Gibt die Art des visuellen Feedbacks an, das Benutzern beim Ziehen eines Verbinders angezeigt wird. Wenn die Maustaste losgelassen wird, wird die resultierende Verbinderform von dieser Einstellung nicht beeinflusst. Dieses Element gilt nicht für routingfähige Konnektoren.

```csharp
public enum DynFeedbackValue
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| RemainStraight | `0` | Bleiben Sie gerade (keine Beine). |
| ShowThreeLegs | `1` | Beim Ziehen drei Beine anzeigen. |
| ShowFiveLegs | `2` | Beim Ziehen fünf Beine anzeigen. |
| Undefined | `-2147483648` | Undefiniert. |
