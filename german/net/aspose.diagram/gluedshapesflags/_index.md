## GluedShapesFlags enumeration

Gibt Konstanten an, die angeben, welche Formen zurückgegeben werden sollen, basierend auf der Dimensionalität und Richtung der Verbindungspunkte, die an eine bestimmte Form geklebt werden; an die Shape.GluedShapes-Methode übergeben.

```csharp
public enum GluedShapesFlags
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| GluedShapesAll1D | `0` | Gibt die IDs aller 1-D-Formen zurück, die an diese Form geklebt werden. |
| GluedShapesIncoming1D | `1` | Gibt IDs von 1D-Formen zurück, deren Endpunkte an diese Form geklebt sind. |
| GluedShapesOutgoing1D | `2` | Gibt IDs von 1D-Formen zurück, deren Anfangspunkte an diese Form geklebt sind. |
| GluedShapesAll2D | `3` | Gibt alle 2D-Formen zurück, die an diese Form geklebt sind, und alle 2D-Formen, an die diese Form geklebt ist. |
| GluedShapesIncoming2D | `4` | Wenn das Quellobjekt eine 1-D-Form ist, geben Sie die IDs der 2-D-Form zurück, an die der Anfangspunkt geklebt ist. Wenn das Quellobjekt eine 2D-Form ist, IDs von 2D-Formen zurückgeben, die an diese Form geklebt sind. |
| GluedShapesOutgoing2D | `5` | Wenn das Quellobjekt eine 1-D-Form ist, geben Sie die IDs der 2-D-Form zurück, an die der Endpunkt geklebt ist. Wenn das Quellobjekt eine 2D-Form ist, IDs von 2D-Formen zurückgeben, an die diese Form geklebt ist. |
