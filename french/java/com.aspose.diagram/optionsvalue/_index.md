Entier non signé facultatif. Options à appliquer à l'ensemble d'enregistrements de données. Les valeurs possibles peuvent être toute combinaison d'une ou plusieurs de celles présentées dans le tableau suivant.

| [NO_ADV_CONFIG](#NO-ADV-CONFIG) | Limite le contrôle des utilisateurs sur la façon dont l'ensemble d'enregistrements de données est actualisé dans la boîte de dialogue Configurer l'actualisation pour l'ensemble d'enregistrements de données. |
| [NO_EXTERNAL_DATA_UI](#NO-EXTERNAL-DATA-UI) | Empêche les données de l'ensemble d'enregistrements de données d'être affichées dans la fenêtre Données externes. |
| [NO_LINK_ON_PASTE](#NO-LINK-ON-PASTE) | Ne copie pas les liens de données de forme dans le presse-papiers lorsque les formes sont copiées ou coupées. |
| [NO_REFRESH_UI](#NO-REFRESH-UI) | Empêche l'ensemble d'enregistrements de données d'être affiché dans la boîte de dialogue Actualiser les données. |

### DELAY_QUERY {#DELAY-QUERY}
```
public static final int DELAY_QUERY
```


N'exécute pas la requête chaîne de commande avant la prochaine actualisation de l'ensemble d'enregistrements de données.

### NO_ADV_CONFIG {#NO-ADV-CONFIG}
```
public static final int NO_ADV_CONFIG
```


Limite le contrôle des utilisateurs sur la façon dont l'ensemble d'enregistrements de données est actualisé dans la boîte de dialogue Configurer l'actualisation pour l'ensemble d'enregistrements de données. En particulier, les utilisateurs ne peuvent pas modifier la clé primaire ni spécifier le moment où les données de forme doivent être écrasées ; cependant, ils peuvent définir l'intervalle d'actualisation et modifier la source de données.

### NO_EXTERNAL_DATA_UI {#NO-EXTERNAL-DATA-UI}
```
public static final int NO_EXTERNAL_DATA_UI
```


Empêche les données de l'ensemble d'enregistrements de données d'être affichées dans la fenêtre Données externes.

### NO_LINK_ON_PASTE {#NO-LINK-ON-PASTE}
```
public static final int NO_LINK_ON_PASTE
```


Ne copie pas les liens de données de forme dans le presse-papiers lorsque les formes sont copiées ou coupées.
