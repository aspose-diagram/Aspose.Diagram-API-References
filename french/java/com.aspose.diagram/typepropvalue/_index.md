### BOOLEAN {#BOOLEAN}
```
public static final int BOOLEAN
```


Booléen. Affiche FALSE et TRUE comme éléments que les utilisateurs peuvent sélectionner dans une liste déroulante de la boîte de dialogue Propriétés personnalisées dans l'application Visio.

### CURRENCY {#CURRENCY}
```
public static final int CURRENCY
```


Valeur monétaire. Utilise les paramètres régionaux actuels du système.

### DATE_TIME {#DATE-TIME}
```
public static final int DATE_TIME
```


Valeur de date ou d'heure. Affiche les jours, mois et années, ou les secondes, minutes et heures, ou une valeur combinée de date et d'heure.

### DURATION {#DURATION}
```
public static final int DURATION
```


Valeur de durée. Affiche le temps écoulé.

### FIXED_LIST {#FIXED-LIST}
```
public static final int FIXED_LIST
```


Liste fixe. Affiche les éléments de la liste dans une boîte combinée déroulante de la boîte de dialogue Propriétés personnalisées.

### NUMBER {#NUMBER}
```
public static final int NUMBER
```


Nombre. Inclut les valeurs de date, d'heure, de durée et monétaires ainsi que les scalaires, dimensions et angles.

### STRING {#STRING}
```
public static final int STRING
```


Chaîne. C'est la valeur par défaut.

### UNDEFINED {#UNDEFINED}
```
public static final int UNDEFINED
```


Indéfini.

### VARIABLE_LIST {#VARIABLE-LIST}
```
public static final int VARIABLE_LIST
```


Liste variable. Affiche les éléments de la liste dans une boîte combinée déroulante de la boîte de dialogue Propriétés personnalisées dans Visio. Les utilisateurs peuvent sélectionner un élément de la liste ou saisir un nouvel élément qui est ajouté à la liste actuelle dans l'élément Format.
