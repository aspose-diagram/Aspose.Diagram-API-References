### fromArgb(int red, int green, int blue) {#fromArgb-int-int-int-}
```
public static Color fromArgb(int red, int green, int blue)
```


Crée un objet Color à partir des valeurs de couleur 8 bits spécifiées (rouge, vert et bleu). La valeur alpha est implicitement 255 (complètement opaque). Bien que cette méthode permette de passer une valeur 32 bits pour chaque composante de couleur, la valeur de chaque composante est limitée à 8 bits.

**Parameters:**
| Paramètre | Type | Description |
| --- | --- | --- |
| red | int | La valeur du composant rouge pour le nouvel objet Color. Les valeurs valides sont de 0 à 255. |
| vert | int | La valeur du composant vert pour le nouvel objet Color. Les valeurs valides sont de 0 à 255. |
| bleu | int | La valeur du composant bleu pour le nouvel objet Color. Les valeurs valides sont de 0 à 255. |

**Returns:**
[Color](../../com.aspose.diagram/color) - The Color object that this method creates.
### fromArgb(int alpha, int red, int green, int blue) {#fromArgb-int-int-int-int-}
```
public static Color fromArgb(int alpha, int red, int green, int blue)
```


Crée un objet Color à partir des quatre valeurs de composant ARGB (alpha, rouge, vert et bleu). Bien que cette méthode permette de passer une valeur de 32 bits pour chaque composant, la valeur de chaque composant est limitée à 8 bits.

**Parameters:**
| Paramètre | Type | Description |
| --- | --- | --- |
| alpha | int | Le composant alpha. Les valeurs valides sont de 0 à 255. |
| red | int | Le composant rouge. Les valeurs valides sont de 0 à 255. |
| vert | int | Le composant vert. Les valeurs valides sont de 0 à 255. |
| bleu | int | Le composant bleu. Les valeurs valides sont de 0 à 255. |

**Returns:**
[Color](../../com.aspose.diagram/color) - The Color object that this method creates.
