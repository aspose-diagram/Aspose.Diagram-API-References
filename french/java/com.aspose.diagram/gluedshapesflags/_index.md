public final class GluedShapesFlags

Spécifie les constantes qui identifient les formes à renvoyer, en fonction de la dimensionalité et de la directionnalité des points de connexion collés à une forme particulière ; transmises à la méthode Shape.GluedShapes.
## Champs

| Champ | Description |
| --- | --- |
| [GLUED_SHAPES_ALL_1_D](#GLUED-SHAPES-ALL-1-D) | Renvoie les ID de toutes les formes 1-D collées à cette forme. |
| [GLUED_SHAPES_ALL_2_D](#GLUED-SHAPES-ALL-2-D) | Renvoie toutes les formes 2-D collées à cette forme ainsi que toutes les formes 2-D auxquelles cette forme est collée. |
| [GLUED_SHAPES_INCOMING_1_D](#GLUED-SHAPES-INCOMING-1-D) | Renvoie les ID des formes 1-D dont les points d'extrémité sont collés à cette forme. |
| [GLUED_SHAPES_INCOMING_2_D](#GLUED-SHAPES-INCOMING-2-D) | Si l'objet source est une forme 1-D, renvoie les ID de la forme 2-D à laquelle le point de départ est collé. |
| [GLUED_SHAPES_OUTGOING_1_D](#GLUED-SHAPES-OUTGOING-1-D) | Retourner les ID des formes 1-D dont les points de début sont collés à cette forme. |
| [GLUED_SHAPES_OUTGOING_2_D](#GLUED-SHAPES-OUTGOING-2-D) | Si l'objet source est une forme 1-D, retourner les ID de la forme 2-D à laquelle le point d'extrémité est collé. |
