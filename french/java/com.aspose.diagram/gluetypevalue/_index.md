java.lang.Object
```
public final class GlueTypeValue
```

Spécifie si la colle dynamique (forme-à-forme) est autorisée lors de la connexion à une forme.
## Champs

| Champ | Description |
| --- | --- |
| [ALLOW_DYNAMIC_GLUE](#ALLOW-DYNAMIC-GLUE) | Autoriser la colle dynamique. |
| [ALLOW_DYNAMIC_GLUE_2002](#ALLOW-DYNAMIC-GLUE-2002) | Autoriser la colle dynamique (obsolète dans Microsoft Visio 2002). |
| [ALLOW_DYNAMIC_GLUE_FOR_DYNAMIC_CONNECTOR](#ALLOW-DYNAMIC-GLUE-FOR-DYNAMIC-CONNECTOR) | Par défaut. |
| [NO_ALLOW_2_D_SHAPE](#NO-ALLOW-2-D-SHAPE) | Ne pas autoriser cette forme 2D d'être connectée en utilisant la colle dynamique. |
| [NO_ALLOW_DYNAMIC_GLUE](#NO-ALLOW-DYNAMIC-GLUE) | Ne pas autoriser la colle dynamique. |
| [UNDEFINED](#UNDEFINED) | Indéfini. |
