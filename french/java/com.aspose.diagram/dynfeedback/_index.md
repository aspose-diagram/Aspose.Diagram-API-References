java.lang.Object
```
public class DynFeedback
```

Spécifie le type de retour visuel fourni aux utilisateurs lorsqu'ils font glisser un connecteur. Lorsque le bouton de la souris est relâché, la forme du connecteur résultante n'est pas affectée par ce paramètre. Cet élément ne s'applique pas aux connecteurs routables.
