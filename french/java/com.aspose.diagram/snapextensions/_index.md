java.lang.Object
```
public final class SnapExtensions
```

Spécifie si un paramètre d'extension d'accrochage spécifique est activé ou désactivé pour la fenêtre active. La valeur peut être la somme des valeurs.
## Champs

| Champ | Description |
| --- | --- |
| [ALIGNMENT_BOX_EXTENSION](#ALIGNMENT-BOX-EXTENSION) | Accrocher à la boîte d'alignement extension. |
| [CENTER_AXES](#CENTER-AXES) | Accrocher à l'axe central extension. |
| [CURVE_EXTENSION](#CURVE-EXTENSION) | Accrocher à la courbe extension. |
| [CURVE_TANGENT](#CURVE-TANGENT) | Accrocher à la tangente de la courbe extension. |
| [ELLIPSE_CENTER](#ELLIPSE-CENTER) | Accrocher au centre de l'ellipse extension. |
| [ENDPOINT](#ENDPOINT) | Accrocher au point final extension. |
| [ENDPOINT_HORIZONTAL](#ENDPOINT-HORIZONTAL) | Accrocher au point final horizontal extension. |
| [ENDPOINT_PERPENDICULAR](#ENDPOINT-PERPENDICULAR) | Accrocher au point final perpendiculaire extension. |
| [ENDPOINT_VERTICAL](#ENDPOINT-VERTICAL) | Accrocher au point final vertical extension. |
| [ISOMETRIC_ANGLES](#ISOMETRIC-ANGLES) | Accrocher aux angles isométriques extension. |
| [LINEAR_EXTENSION](#LINEAR-EXTENSION) | Accrocher à l'extension linéaire. |
| [MIDPOINT](#MIDPOINT) | Accrocher au point médian extension. |
| [MIDPOINT_PERPENDICULAR](#MIDPOINT-PERPENDICULAR) | Accrocher au point médian perpendiculaire extension. |
| [NONE](#NONE) | Accrocher à rien. |
| [UNDEFINED](#UNDEFINED) | Indéfini. |
