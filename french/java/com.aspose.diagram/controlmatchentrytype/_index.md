Représente la façon dont une ListBox ou une ComboBox recherche dans sa liste pendant que l'utilisateur tape.
## Champs

| Champ | Description |
| --- | --- |
| [COMPLETE](#COMPLETE) | À chaque caractère tapé, le contrôle recherche une entrée correspondant à tous les caractères saisis. |
| [FIRST_LETTER](#FIRST-LETTER) | Le contrôle recherche l'entrée suivante qui commence par le caractère saisi. |
| [NONE](#NONE) | La liste ne sera pas recherchée lorsque des caractères sont tapés. |

### COMPLETE {#COMPLETE}
```
public static final int COMPLETE
```


À chaque caractère tapé, le contrôle recherche une entrée correspondant à tous les caractères saisis.

### FIRST_LETTER {#FIRST-LETTER}
```
public static final int FIRST_LETTER
```


Le contrôle recherche l'entrée suivante qui commence par le caractère saisi. Taper à plusieurs reprises la même lettre fait défiler toutes les entrées commençant par cette lettre.

### NONE {#NONE}
```
public static final int NONE
```


La liste ne sera pas recherchée lorsque des caractères sont tapés.
