### DEPENDS_ON_DRAWING_SIZE_TYPE {#DEPENDS-ON-DRAWING-SIZE-TYPE}
```
public static final int DEPENDS_ON_DRAWING_SIZE_TYPE
```


Le fait que la page se redimensionne automatiquement dépend de la valeur de la cellule DrawingSizeType. Si DrawingSizeType = 0 (la taille de la page de dessin est identique à celle de la page imprimée), la page se redimensionne automatiquement. Si la valeur de DrawingSizeType est différente de zéro (0), la page ne se redimensionne pas automatiquement.
