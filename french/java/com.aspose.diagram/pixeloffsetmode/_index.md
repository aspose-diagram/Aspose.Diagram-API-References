Spécifie comment les pixels sont décalés lors du rendu.
## Champs

| Champ | Description |
| --- | --- |
| [DEFAULT](#DEFAULT) | Spécifie le mode par défaut. |
| [HALF](#HALF) | Spécifie que les pixels sont décalés de -0,5 unités, à la fois horizontalement et verticalement, pour un antialiasing à grande vitesse. |
| [HIGH_QUALITY](#HIGH-QUALITY) | Spécifie un rendu de haute qualité, à basse vitesse. |
| [HIGH_SPEED](#HIGH-SPEED) | Spécifie un rendu à haute vitesse, de basse qualité. |
| [INVALID](#INVALID) | Spécifie un mode invalide. |
| [NONE](#NONE) | Spécifie aucun décalage de pixel. |
