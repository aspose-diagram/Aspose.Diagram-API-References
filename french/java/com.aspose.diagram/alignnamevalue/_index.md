int optionnel. Spécifie si le texte du maître dans la fenêtre de pochoir est aligné à gauche, à droite ou au centre.
## Champs

| Champ | Description |
| --- | --- |
| [ALIGN_TEXT_CENTER](#ALIGN-TEXT-CENTER) | Aligner le texte au centre. |
| [ALIGN_TEXT_LEFT](#ALIGN-TEXT-LEFT) | Aligner le texte à gauche. |
| [ALIGN_TEXT_RIGHT](#ALIGN-TEXT-RIGHT) | Aligner le texte à droite. |
