```
public final class WindowTypeValue
```

Une valeur énumérée qui peut être l'une des suivantes: Drawing, Sheet, Stencil ou Icon. Un élément Window de WindowType='Stencil' doit apparaître après son fenêtre parent de dessin (WindowType='Drawing') et avant tout autre élément de fenêtre de dessin.
