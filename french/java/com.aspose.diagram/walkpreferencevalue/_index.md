Spécifie si une extrémité d'une forme 1D se déplace vers un point de connexion horizontal ou vertical sur la forme à laquelle elle est collée, en utilisant la colle dynamique, lorsque la forme est déplacée vers une position ambiguë.
## Champs

| Champ | Description |
| --- | --- |
| [SIDE_TO_SIDE_CONNECTIONS](#SIDE-TO-SIDE-CONNECTIONS) | La valeur par défaut. |
| [SIDE_TO_TOP_OR_SIDE_TO_BOTTOM_CONNECTIONS](#SIDE-TO-TOP-OR-SIDE-TO-BOTTOM-CONNECTIONS) | Le point de départ de la forme 1-D se déplace vers un point de connexion horizontal, et le point d'arrivée se déplace vers un point de connexion vertical (connexions côté‑vers‑haut ou côté‑vers‑bas). |
| [TOP_TO_BOTTOM_CONNECTIONS](#TOP-TO-BOTTOM-CONNECTIONS) | Les deux extrémités de la forme 1-D se déplacent vers des points de connexion verticaux (connexions haut‑vers‑bas). |
| [TOP_TO_SIDE_OR_BOTTOM_TO_SIDE_CONNECTIONS](#TOP-TO-SIDE-OR-BOTTOM-TO-SIDE-CONNECTIONS) | Le point de départ de la forme 1-D se déplace vers un point de connexion vertical, et le point d'arrivée se déplace vers un point de connexion horizontal (connexions haut‑vers‑côté ou bas‑vers‑côté). |
| [UNDEFINED](#UNDEFINED) | Indéfini. |

### SIDE_TO_SIDE_CONNECTIONS {#SIDE-TO-SIDE-CONNECTIONS}

Par défaut. Les deux extrémités de la forme 1-D se déplacent vers des points de connexion horizontaux (connexions côté‑vers‑côté).

### SIDE_TO_TOP_OR_SIDE_TO_BOTTOM_CONNECTIONS {#SIDE-TO-TOP-OR-SIDE-TO-BOTTOM-CONNECTIONS}
