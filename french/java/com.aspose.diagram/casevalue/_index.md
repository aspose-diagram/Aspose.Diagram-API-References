```
public final class CaseValue
```

Détermine la casse du texte d'une forme. Toutes les lettres majuscules (uppercase) (1) et les majuscules initiales (2) ne modifient pas l'apparence du texte saisi en majuscules. Le texte doit être saisi en minuscules pour que ces options produisent un effet.
## Champs

| Champ | Description |
| --- | --- |
| [ALL_CAPITAL_LETTERS](#ALL-CAPITAL-LETTERS) | Toutes les lettres en majuscules (uppercase). |
| [INITIAL_CAPITAL_LETTERS_ONLY](#INITIAL-CAPITAL-LETTERS-ONLY) | Seules les lettres initiales en majuscules. |
| [NORMAL_CASE](#NORMAL-CASE) | Casse normale. |
| [UNDEFINED](#UNDEFINED) | Indéfini. |
