Les valeurs binaires indiquent qu'un paramètre de colle spécifique est activé ou désactivé. La valeur peut être la somme des valeurs :
## Champs

| Champ | Description |
| --- | --- |
| [CONNECTION_POINTS](#CONNECTION-POINTS) | Coller aux points de connexion. |
| [DISABLED](#DISABLED) | La colle est désactivée |
| [GEOMETRY](#GEOMETRY) | Coller à la géométrie. |
| [GUIDES](#GUIDES) | Coller aux repères. |
| [HANDLES](#HANDLES) | Coller aux poignées. |
| [NONE](#NONE) | Le collage est activé, mais aucun autre paramètre de collage n’est activé. |
| [UNDEFINED](#UNDEFINED) | Indéfini. |
| [VERTICES](#VERTICES) | Coller aux sommets. |
