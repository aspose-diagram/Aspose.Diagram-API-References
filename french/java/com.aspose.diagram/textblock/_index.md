### getBottomMargin() {#getBottomMargin--}
```
public DoubleValue getBottomMargin()
```


Détermine la distance entre la bordure inférieure du bloc de texte et la dernière ligne de texte qu'il contient. La valeur par défaut est de 4 pt. Cette valeur est indépendante de l'échelle du dessin. Si le dessin est mis à l'échelle, la marge inférieure reste la même.
