public final class PlaceFlipValue

Spécifie comment les formes placables sont retournées et/ou pivotées sur une page lorsque les formes sont disposées à l'aide de la commande Lay Out Shapes dans Microsoft Visio. Les valeurs hexadécimales suivantes sont autorisées.
## Champs

| Champ | Description |
| --- | --- |
| [DEFAULT_NO_FLIP](#DEFAULT-NO-FLIP) | Par défaut. |
| [FLIP_90_INCREMENTS](#FLIP-90-INCREMENTS) | Retourner par incréments de 90 degrés. |
| [FLIP_HORIZONTAL](#FLIP-HORIZONTAL) | Retourner horizontalement. |
| [FLIP_VERTICAL](#FLIP-VERTICAL) | Retourner verticalement. |
| [NO_FLIP](#NO-FLIP) | Pas de retournement. |
| [UNDEFINED](#UNDEFINED) | Indéfini. |
