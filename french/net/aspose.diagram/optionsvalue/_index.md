## OptionsValue enumeration

Entier non signé facultatif. Options à appliquer au jeu d'enregistrements de données. Les valeurs possibles peuvent être n'importe quelle combinaison d'une ou plusieurs de celles indiquées dans le tableau suivant.

```csharp
[Flags]
public enum OptionsValue
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| NoExternalDataUI | `1` | Empêche l'affichage des données du jeu d'enregistrements de données dans la fenêtre Données externes. |
| NoAdvConfig | `4` | Limite le contrôle dont disposent les utilisateurs sur la façon dont le jeu d'enregistrements de données est actualisé dans la boîte de dialogue Configurer l'actualisation pour le jeu d'enregistrements de données.
