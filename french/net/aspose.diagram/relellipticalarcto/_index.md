## RelEllipticalArcTo class

| Nom | La description |

| [A](../../aspose.diagram/relellipticalarcto/a) { get; set; } | La coordonnée x du point de contrôle de l'arc. Sinon, l'arc peut atteindre une taille extrême afin de passer par le point de contrôle, avec des résultats imprévisibles. |

| [D](../../aspose.diagram/relellipticalarcto/d) { get; set; } | Le rapport du grand axe d'un arc à son petit axe. Malgré le sens habituel de ces mots, le grand axe n'a pas à être supérieur au petit axe, ce rapport n'a donc pas à être supérieur à 1. Réglage de cet élément à une valeur inférieure ou égale à 0 ou supérieure à 1000 peut conduire à des résultats imprévisibles. |
