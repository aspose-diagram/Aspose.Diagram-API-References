## DynFeedbackValue enumeration

Spécifie le type de retour visuel fourni aux utilisateurs lorsqu'ils font glisser un connecteur. Lorsque le bouton de la souris est relâché, la forme du connecteur résultant n'est pas affectée par ce paramètre. Cet élément ne s'applique pas aux connecteurs routables.

```csharp
public enum DynFeedbackValue
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| RemainStraight | `0` | Restez droit (pas de jambes). |
| ShowThreeLegs | `1` | Afficher trois jambes lorsqu'il est déplacé. |
| ShowFiveLegs | `2` | Afficher cinq jambes lorsqu'il est déplacé. |
| Undefined | `-2147483648` | Non défini. |
