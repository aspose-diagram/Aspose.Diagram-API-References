## Prop.Prompt property

L'élément d'invite spécifie le texte descriptif ou d'instruction qui s'affiche pour les utilisateurs dans la boîte de dialogue Propriétés personnalisées lorsque la propriété est sélectionnée. Ce texte apparaît également sous forme d'info-bulle lorsque le pointeur de la souris est placé sur la propriété dans la fenêtre Propriétés personnalisées.
