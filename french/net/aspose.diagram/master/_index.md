## Master class

Contient des éléments qui définissent un maître pour le document. Une forme de base est une forme sur un gabarit que vous utilisez à plusieurs reprises pour créer des dessins. Lorsque vous faites glisser une forme d'un gabarit vers la page de dessin, la forme devient une instance de cette forme de base et une copie locale de la forme de base est incluse dans le document.

```csharp
public class Master : IDisposable
```
