## Master.MatchByName property

L'attribut MatchByName détermine comment Microsoft Visio détermine si un masque de document est déjà présent lorsqu'une instance d'un masque est déposée sur la page de dessin. Il permet aux modifications apportées à un maître de document de s'appliquer aux nouvelles instances du maître, même si les instances sont glissées à partir d'un fichier de gabarit autonome.
