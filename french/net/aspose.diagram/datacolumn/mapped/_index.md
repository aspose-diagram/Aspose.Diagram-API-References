## DataColumn.Mapped property

Spécifie si la colonne est visible dans la fenêtre Données externes. Vrai (1) pour que la colonne soit visible ; false (0) pour que la colonne ne soit pas visible. La valeur par défaut (attribut absent) est que la colonne soit visible.
