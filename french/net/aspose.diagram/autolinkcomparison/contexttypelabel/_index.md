## AutoLinkComparison.ContextTypeLabel property

Si la valeur ContextType est 2 ou 3, cet attribut est requis pour définir une comparaison. Pour ContextType = 2, ContextTypeLabel doit être l'étiquette de l'élément de données de forme, et si ContextType = 3, ContextTypeLabel doit être le nom de la ligne locale.
