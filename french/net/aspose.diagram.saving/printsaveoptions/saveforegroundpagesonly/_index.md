## PrintSaveOptions.SaveForegroundPagesOnly property

Spécifie si toutes les pages seront imprimées ou uniquement au premier plan.

```csharp
public bool SaveForegroundPagesOnly { get; set; }
```

### Remarques

Si vrai - imprimé uniquement les pages de premier plan (avec arrière-plan si présent). Si faux - imprimé les pages de premier plan (avec arrière-plan si présent) après ces pages d'arrière-plan vides. Peut renvoyer vrai uniquement lorsque PageCount &gt; 1. La valeur par défaut est faux .
