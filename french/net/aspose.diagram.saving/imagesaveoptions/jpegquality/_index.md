## ImageSaveOptions.JpegQuality property

Obtient ou définit une valeur déterminant la qualité des images JPEG générées.

```csharp
public int JpegQuality { get; set; }
```

### Remarques

N'a d'effet que lors de l'enregistrement au format JPEG. Utilisez cette propriété pour obtenir ou définir la qualité des images générées lors de l'enregistrement au format JPEG. La valeur peut varier de 0 à 100, où 0 signifie la pire qualité mais la compression maximale et 100 signifie la meilleure qualité mais compression minimum. La valeur par défaut est 95.
