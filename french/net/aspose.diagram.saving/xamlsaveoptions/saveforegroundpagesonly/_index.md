## XAMLSaveOptions.SaveForegroundPagesOnly property

Spécifie si toutes les pages seront enregistrées en image ou uniquement au premier plan.

```csharp
public bool SaveForegroundPagesOnly { get; set; }
```

### Remarques

Si vrai - rendu uniquement les pages de premier plan (avec arrière-plan si présent). Si faux - rendu les pages de premier plan (avec arrière-plan si présent) après ces pages d'arrière-plan vides. Peut renvoyer vrai uniquement lorsque PageCount &gt; 1. La valeur par défaut est faux .
