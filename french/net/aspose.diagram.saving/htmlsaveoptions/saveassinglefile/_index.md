## HTMLSaveOptions.SaveAsSingleFile property

Indique si enregistrer le HTML en tant que fichier unique. La valeur par défaut est false.

```csharp
public bool SaveAsSingleFile { get; set; }
```

### Remarques

S'il y a plusieurs pages, ces pages et autres ressources doivent être enregistrées dans des fichiers séparés. Pour certains scénarios, l'utilisateur peut avoir besoin d'obtenir un seul fichier résultant, par exemple pour faciliter le transfert. Si c'est le cas, l'utilisateur peut définir cette propriété sur true.
