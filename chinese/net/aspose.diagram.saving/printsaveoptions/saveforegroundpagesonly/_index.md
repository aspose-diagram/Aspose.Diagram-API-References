## PrintSaveOptions.SaveForegroundPagesOnly property

指定是打印所有页面还是只打印前景。

```csharp
public bool SaveForegroundPagesOnly { get; set; }
```

### 评论

如果为 true - 仅打印前景页面（如果存在则有背景）。 如果为 false - 在空背景页面之后打印前景页面（如果存在则有背景）。 只有当 PageCount &gt; 1 时才能返回 true。 默认值为 false .
