## PrintSaveOptions.SaveForegroundPagesOnly property

指定是打印所有页面还是仅打印前景。

```csharp
public bool SaveForegroundPagesOnly { get; set; }
```

### 评论

如果为真 - 仅打印前景页面（如果存在背景）。 If false - 在空的背景页面之后打印前景页面（如果存在背景）。 只有当 PageCount &gt; 时才能返回 true 1. 默认值为false。
