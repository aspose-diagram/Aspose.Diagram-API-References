## RenderingSaveOptions.EmfRenderSetting property

渲染 Emf 图元文件的设置。

```csharp
public EmfRenderSetting EmfRenderSetting { get; set; }
```

### 评论

标识为“EMF+ Dual”的 EMF 元文件可以同时包含 EMF+ 记录和 EMF 记录。 任何一种类型的记录都可用于渲染图像，仅 EMF+ 记录或仅 EMF 记录。 当设置!:EmfRenderSetting.EmfPlusPrefer时，将解析 EMF+ 记录，否则仅解析 EMF 记录。 默认值为!:EmfRenderSetting.EmfOnly。
