## XPSSaveOptions.SaveForegroundPagesOnly property

指定所有页面将保存在图像中还是仅在前景中。

```csharp
public bool SaveForegroundPagesOnly { get; set; }
```

### 评论

如果为 true - 仅呈现前台页面（如果存在则显示背景）。 If false - 在空的背景页面之后呈现前景页面（如果存在背景）。 只有当 PageCount &gt; 时才能返回 true 1. 默认值为false。
