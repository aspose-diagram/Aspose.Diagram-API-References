## PdfPermissions enumeration

指定 PDF 文档的用户权限。

```csharp
[Flags]
public enum PdfPermissions
```

### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| DisallowAll | `0` | 禁止对 PDF 文档进行所有操作。这是默认值。 |
| Printing | `4` | 允许打印文档。 |
| ModifyContents | `8` | 允许修改文档的内容。 |
| ContentCopy | `10` | 允许从文档中复制或提取文本和图形，包括出于可访问性目的提取。 |
| ModifyAnnotations | `20` | 允许添加或修改文本注释。使用 RC4 40 位加密时，此选项还允许填写表单字段。 |
| FillIn | `100` | 允许填写表格和签署文件。使用 RC4 40 位加密时，此选项将被忽略，并且只要设置了 ModifyAnnotations，就允许填写表单。 |
| ContentCopyForAccessibility | `200` | 允许提取文本和图形以支持残疾用户的可访问性或用于其他目的。使用 RC4 40 位加密时，此选项将被忽略，并且只要设置了 ContentCopy，就允许访问。 |
| DocumentAssembly | `400` | 允许组装文档:插入、旋转或删除页面以及创建导航元素，例如书签或缩略图。使用 RC4 40 位加密时，此选项将被忽略，并且在设置 ModifyContents 时允许文档汇编。 |
| HighResolutionPrinting | `804` | 允许以可能的最高分辨率打印文档。使用 RC4 40 位加密时，忽略此选项，设置打印时允许高分辨率打印。 |
| AllowAll | `FFFF` | 允许对 PDF 文档进行所有操作。 |
