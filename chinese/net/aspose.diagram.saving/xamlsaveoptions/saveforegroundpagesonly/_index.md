## XAMLSaveOptions.SaveForegroundPagesOnly property

指定是将所有页面保存在图像中还是仅保存在前景中。

```csharp
public bool SaveForegroundPagesOnly { get; set; }
```

### 评论

如果为 true - 仅呈现前景页面（如果存在则带背景）。 如果为 false - 在空白背景页面之后呈现前景页面（如果存在则带背景）。 只有当 PageCount &gt; 1 时才能返回 true。 默认值为 false .
