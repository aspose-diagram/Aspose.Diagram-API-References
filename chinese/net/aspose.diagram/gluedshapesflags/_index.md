## GluedShapesFlags enumeration

指定常量，根据粘附到特定形状的连接点的维数和方向来标识要返回的形状；传递给 Shape.GluedShapes 方法。

```csharp
public enum GluedShapesFlags
```

| 姓名 | 价值 | 描述 |

| GluedShapesAll1D | `0` | 返回粘附到此形状的所有一维形状的 ID。 |
| GluedShapesIncoming1D | `1` | 返回端点粘附到此形状的一维形状的 ID。 |
