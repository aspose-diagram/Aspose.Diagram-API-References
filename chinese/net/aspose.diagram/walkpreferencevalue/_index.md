## WalkPreferenceValue enumeration

指定当形状移动时，使用动态粘合，一维形状的端点是否移动到它所粘合的形状上的水平或垂直连接点到一个模棱两可的位置。

```csharp
public enum WalkPreferenceValue
```

| 姓名 | 价值 | 描述 |

| SideToSideConnections | `0` | 默认值。一维形状的两个端点都移动到水平连接点（左右连接）。 |
