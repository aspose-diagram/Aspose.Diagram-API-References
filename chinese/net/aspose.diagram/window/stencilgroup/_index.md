## Window.StencilGroup property

指定窗口所属的合并模板窗口组。此属性仅与 WindowType 属性为 Stencil 的 Window 元素相关，并且仅当模板窗口是模板窗口合并组的一部分时才相关。属于同一合并组的所有模板窗口都具有相同的 StencilGroup 元素值。

public string StencilGroup { get; set; }
