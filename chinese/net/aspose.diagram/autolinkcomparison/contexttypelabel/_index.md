## AutoLinkComparison.ContextTypeLabel property

如果 ContextType 值为 2 或 3，则需要此属性来定义比较。对于ContextType = 2，ContextTypeLabel必须是形状数据项标签，如果ContextType = 3，ContextTypeLabel必须是本地行名.

public string ContextTypeLabel { get; set; }
