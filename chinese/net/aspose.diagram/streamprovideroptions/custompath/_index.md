## StreamProviderOptions.CustomPath property

用户自定义路径 (URL) 保存在生成的 html 文件中，用于引用源。如果用户未定义，则使用默认路径。 例如，用户将工作表数据保存到 d:/sheet001.htm，主 html 文件中使用的 url 应为“d:/sheet001.htm”或主 html 文件可以访问的其他有效相对路径.
