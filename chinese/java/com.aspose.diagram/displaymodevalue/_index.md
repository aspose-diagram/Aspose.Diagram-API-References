java.lang.Object
```
public final class DisplayModeValue
```

当它位于 Group 元素中时，DisplayMode 元素指定组形状及其成员的显示方式。当它位于 SmartTagDef 元素中时，DisplayMode 元素决定智能标签是在用户将鼠标悬停在标签上时出现、在形状被选中时出现，还是始终出现。

| [DISPLAYS_SHAPE_BEHIND_MEMBER_SHAPES](#DISPLAYS-SHAPE-BEHIND-MEMBER-SHAPES) | 在成员形状后面显示组形状。 |
| [DISPLAYS_SHAPE_FRONT_MEMBER_SHAPES](#DISPLAYS-SHAPE-FRONT-MEMBER-SHAPES) | 在成员形状前面显示组形状。 |
| [HIDES_SHAPE_TEXT](#HIDES-SHAPE-TEXT) | 隐藏组形状和文本。 |
