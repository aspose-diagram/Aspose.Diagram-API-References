**Inheritance:**
java.lang.Object
```
public final class SnapExtensions
```

指定特定的捕捉扩展设置在活动窗口中是启用还是禁用。该值可以是多个值的总和。
## 字段

| 字段 | 描述 |
| --- | --- |
| [ALIGNMENT_BOX_EXTENSION](#ALIGNMENT-BOX-EXTENSION) | 捕捉对齐框扩展。 |
| [CENTER_AXES](#CENTER-AXES) | 捕捉中心轴扩展。 |
| [CURVE_EXTENSION](#CURVE-EXTENSION) | 捕捉曲线扩展。 |
| [CURVE_TANGENT](#CURVE-TANGENT) | 捕捉曲线切线扩展。 |
| [ELLIPSE_CENTER](#ELLIPSE-CENTER) | 捕捉椭圆中心扩展。 |
| [ENDPOINT](#ENDPOINT) | 捕捉端点扩展。 |
| [ENDPOINT_HORIZONTAL](#ENDPOINT-HORIZONTAL) | 捕捉端点水平扩展。 |
| [ENDPOINT_PERPENDICULAR](#ENDPOINT-PERPENDICULAR) | 捕捉端点正交扩展。 |
| [ENDPOINT_VERTICAL](#ENDPOINT-VERTICAL) | 捕捉端点垂直扩展。 |
| [ISOMETRIC_ANGLES](#ISOMETRIC-ANGLES) | 捕捉等轴角度扩展。 |
| [LINEAR_EXTENSION](#LINEAR-EXTENSION) | 捕捉线性扩展。 |
| [MIDPOINT](#MIDPOINT) | 捕捉中点扩展。 |
| [MIDPOINT_PERPENDICULAR](#MIDPOINT-PERPENDICULAR) | 捕捉中点正交扩展。 |
| [NONE](#NONE) | 不进行捕捉。 |
| [UNDEFINED](#UNDEFINED) | 未定义。 |
