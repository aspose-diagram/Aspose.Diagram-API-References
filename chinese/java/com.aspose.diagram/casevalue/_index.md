确定形状文本的大小写。全部大写（大写字母）(1) 和首字母大写 (2) 不会改变已全部大写输入的文本外观。必须以小写字母输入文本，才能看到这些选项的效果。
## 字段

| 字段 | 描述 |
| --- | --- |
| [ALL_CAPITAL_LETTERS](#ALL-CAPITAL-LETTERS) | 全部大写（uppercase）字母。 |
| [INITIAL_CAPITAL_LETTERS_ONLY](#INITIAL-CAPITAL-LETTERS-ONLY) | 仅首字母大写。 |
| [NORMAL_CASE](#NORMAL-CASE) | 普通大小写。 |
| [UNDEFINED](#UNDEFINED) | 未定义。 |
