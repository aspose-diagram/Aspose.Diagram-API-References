### fromArgb(int red, int green, int blue) {#fromArgb-int-int-int-}
```
public static Color fromArgb(int red, int green, int blue)
```


从指定的 8 位颜色值（red、green 和 blue）创建一个 Color 对象。alpha 值隐式为 255（完全不透明）。虽然此方法允许为每个颜色分量传递 32 位值，但每个分量的值仅限于 8 位。

**Parameters:**
| 参数 | 类型 | 描述 |
| --- | --- | --- |
| red | int | 新 Color 对象的 red 分量值。有效值范围为 0 到 255。 |
| 绿色 | int | 新 Color 对象的绿色分量值。有效值范围为 0 到 255。 |
| 蓝色 | int | 新 Color 对象的蓝色分量值。有效值范围为 0 到 255。 |

**Returns:**
[Color](../../com.aspose.diagram/color) - The Color object that this method creates.

**Parameters:**
| 参数 | 类型 | 描述 |

| red | int | 红色分量。有效值范围为 0 到 255。 |
| 绿色 | int | 绿色分量。有效值范围为 0 到 255。 |
| 蓝色 | int | 蓝色分量。有效值范围为 0 到 255。 |
