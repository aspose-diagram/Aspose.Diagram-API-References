### DEPENDS_ON_DRAWING_SIZE_TYPE {#DEPENDS-ON-DRAWING-SIZE-TYPE}
```
public static final int DEPENDS_ON_DRAWING_SIZE_TYPE
```


页面是否自动调整大小取决于 DrawingSizeType 单元格的值。如果 DrawingSizeType = 0（绘图页面大小与打印页面大小相同），页面会自动调整大小。如果 DrawingSizeType 的值不是零（0），页面则不会自动调整大小。
