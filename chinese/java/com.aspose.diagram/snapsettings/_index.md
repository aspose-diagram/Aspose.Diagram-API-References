指定当窗口中启用捕捉时，形状捕捉到的对象。该值可能是多个值的和。
## 字段

| 字段 | 描述 |
| --- | --- |
| [ALIGNMENT_BOX](#ALIGNMENT-BOX) | 捕捉到对齐框。 |
| [CONNECTION_POINTS](#CONNECTION-POINTS) | 捕捉到连接点。 |
| [DISABLED](#DISABLED) | 捕捉已禁用。 |
| [EXTENSIONS](#EXTENSIONS) | 捕捉到形状扩展选项。 |
| [GEOMETRY](#GEOMETRY) | 捕捉到形状的可见边缘。 |
| [GRID](#GRID) | 捕捉到网格。 |
| [GUIDES](#GUIDES) | 捕捉到参考线。 |
| [HANDLES](#HANDLES) | 捕捉到选择手柄。 |
| [INTERSECTIONS](#INTERSECTIONS) | 捕捉到交叉点。 |
| [NONE](#NONE) | 不进行捕捉。 |
| [RULER_SUBDIVISIONS](#RULER-SUBDIVISIONS) | 捕捉到标尺细分。 |
| [UNDEFINED](#UNDEFINED) | 未定义。 |
| [VERTICES](#VERTICES) | 捕捉到顶点。 |
