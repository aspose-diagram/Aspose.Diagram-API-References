指定当形状移动到模糊位置时，使用动态粘连时，1-D 形状的端点是否移动到其粘连形状的水平或垂直连接点。
## 字段

| 字段 | 描述 |
| --- | --- |
| [SIDE_TO_SIDE_CONNECTIONS](#SIDE-TO-SIDE-CONNECTIONS) | 默认。 |
| [SIDE_TO_TOP_OR_SIDE_TO_BOTTOM_CONNECTIONS](#SIDE-TO-TOP-OR-SIDE-TO-BOTTOM-CONNECTIONS) | 1-D 形状的起始点移动到水平连接点，终点移动到垂直连接点（侧到上或侧到下的连接）。 |
| [TOP_TO_BOTTOM_CONNECTIONS](#TOP-TO-BOTTOM-CONNECTIONS) | 1-D 形状的两个端点都移动到垂直连接点（上到下的连接）。 |
| [TOP_TO_SIDE_OR_BOTTOM_TO_SIDE_CONNECTIONS](#TOP-TO-SIDE-OR-BOTTOM-TO-SIDE-CONNECTIONS) | 1-D 形状的起始点移动到垂直连接点，终点移动到水平连接点（上到侧或下到侧的连接）。 |
| [UNDEFINED](#UNDEFINED) | 未定义。 |
