java.lang.Object
```
public class XForm
```

包含控制形状线属性的元素，例如图案、粗细和颜色。这些元素决定线端是否格式化（例如使用箭头），线端格式的大小，应用于线的圆角半径，以及线帽样式（圆形或方形）。
