位值指示特定粘合设置是开启还是关闭。该值可能是多个值的总和：
## 字段

| 字段 | 描述 |
| --- | --- |
| [CONNECTION_POINTS](#CONNECTION-POINTS) | 粘合到连接点。 |
| [DISABLED](#DISABLED) | 粘合已禁用 |
| [GEOMETRY](#GEOMETRY) | 粘合到几何体。 |
| [GUIDES](#GUIDES) | 粘合到参考线。 |
| [HANDLES](#HANDLES) | 粘合到手柄。 |
| [NONE](#NONE) | Glue 已启用，但未启用其他粘合设置。 |
| [UNDEFINED](#UNDEFINED) | 未定义。 |
| [VERTICES](#VERTICES) | 粘合到顶点。 |
