```
public final class CompressionType
```

仅当外部数据是基于光栅的外部对象（例如 DIB、JPG、PNG、TIFF 或 GIF 文件）时，此属性才有意义。该值表示对文件应用的压缩类型。
## 字段

| 字段 | 描述 |
| --- | --- |
| [GIF](#GIF) | GIF 压缩。 |
| [JPEG](#JPEG) | JPEG 压缩。 |
| [NO](#NO) | 无压缩（默认）。 |
| [PNG](#PNG) | PNG 压缩。 |
| [TIFF](#TIFF) | TIFF 压缩。 |
| [UNDEFINED](#UNDEFINED) | 未定义。 |
