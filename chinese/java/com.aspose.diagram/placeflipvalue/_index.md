指定在使用 Microsoft Visio 的“布局形状”命令布置形状时，可放置形状在页面上如何翻转和/或旋转。允许使用以下十六进制值。
## 字段

| 字段 | 描述 |
| --- | --- |
| [DEFAULT_NO_FLIP](#DEFAULT-NO-FLIP) | Default. |
| [FLIP_90_INCREMENTS](#FLIP-90-INCREMENTS) | 以 90 度递增翻转。 |
| [FLIP_HORIZONTAL](#FLIP-HORIZONTAL) | 水平翻转。 |
| [FLIP_VERTICAL](#FLIP-VERTICAL) | 垂直翻转。 |
| [NO_FLIP](#NO-FLIP) | 不翻转。 |
| [UNDEFINED](#UNDEFINED) | 未定义。 |
