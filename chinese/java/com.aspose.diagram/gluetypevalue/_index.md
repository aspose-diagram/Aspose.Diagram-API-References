指定在连接到形状时是否允许动态（形状到形状）粘连。
## 字段

| 字段 | 描述 |
| --- | --- |
| [ALLOW_DYNAMIC_GLUE](#ALLOW-DYNAMIC-GLUE) | 允许动态粘合。 |
| [ALLOW_DYNAMIC_GLUE_2002](#ALLOW-DYNAMIC-GLUE-2002) | 允许动态粘合（在 Microsoft Visio 2002 中已废弃）。 |
| [ALLOW_DYNAMIC_GLUE_FOR_DYNAMIC_CONNECTOR](#ALLOW-DYNAMIC-GLUE-FOR-DYNAMIC-CONNECTOR) | Default. |
| [NO_ALLOW_2_D_SHAPE](#NO-ALLOW-2-D-SHAPE) | 不允许使用动态粘合将此二维形状连接到其他形状。 |
| [NO_ALLOW_DYNAMIC_GLUE](#NO-ALLOW-DYNAMIC-GLUE) | 不允许动态粘合。 |
| [UNDEFINED](#UNDEFINED) | 未定义。 |
