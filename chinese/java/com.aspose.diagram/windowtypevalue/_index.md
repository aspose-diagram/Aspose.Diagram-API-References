public final class WindowTypeValue

一个枚举值，可能是以下之一：Drawing、Sheet、Stencil 或 Icon。WindowType='Stencil' 的 Window 元素必须在其父绘图窗口 (WindowType='Drawing') 之后、在任何其他绘图窗口元素之前出现。
