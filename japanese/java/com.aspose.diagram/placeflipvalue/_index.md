| --- | --- |

| [FLIP_90_INCREMENTS](#FLIP-90-INCREMENTS) | 90度単位で反転します。 |
| [FLIP_HORIZONTAL](#FLIP-HORIZONTAL) | 水平に反転します。 |
| [FLIP_VERTICAL](#FLIP-VERTICAL) | 垂直に反転します。 |
| [NO_FLIP](#NO-FLIP) | 反転しません。 |
| [UNDEFINED](#UNDEFINED) | Undefined. |
