java.lang.Object

シェイプのテキストの大文字小文字を決定します。すべて大文字（uppercase）文字（1）および先頭大文字（2）は、すべて大文字で入力されたテキストの外観を変更しません。これらのオプションが効果を示すには、テキストを小文字で入力する必要があります。
