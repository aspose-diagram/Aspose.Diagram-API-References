```
public String getContextTypeLabel()
```


ContextType の値が 2 または 3 の場合、この属性は比較を定義するために必須です。ContextType = 2 の場合、ContextTypeLabel はシェイプ データ アイテムのラベルでなければならず、ContextType = 3 の場合、ContextTypeLabel はローカル行名でなければなりません。
