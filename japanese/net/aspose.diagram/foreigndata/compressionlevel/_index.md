## ForeignData.CompressionLevel property

この属性は、外部データがラスターベースの外部オブジェクト (DIB、JPG、PNG、TIFF、または GIF ファイルなど) である場合にのみ意味があります。この値は、ファイルに適用される圧縮のレベルを示します。圧縮レベルは数百パーセント単位で測定されます.

public double CompressionLevel { get; set; }
