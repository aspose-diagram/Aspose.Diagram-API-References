## ToImage(string, ImageSaveOptions) {#toimage_1}

形状イメージを作成し、ファイルに保存します。 ファイル名の拡張子によって、イメージの形式が決まります。

```csharp
public void ToImage(string imageFile, ImageSaveOptions options)
```

| パラメータ | タイプ | 説明 |
| --- | --- | --- |
| imageFile | String | フルパス付きの画像ファイル名。 |
| options | ImageSaveOptions | 追加の画像作成オプション |

### 備考

画像の形式は、ファイル名の拡張子で指定します。 例えば「myfile.png」と指定すると、画像はPNG形式で で保存されます。次のファイル拡張子が認識されます: .bmp、.gif、.png、.jpg、.jpeg、.tiff、.tif、.emf。
