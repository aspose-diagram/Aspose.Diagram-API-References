## StreamProviderOptions.CustomPath property

参照元の生成された html ファイルに保存されたユーザー カスタム パス (URL)。ユーザーによって定義されていない場合、DefaultPath が使用されます。 たとえば、シート データはユーザーによって d:/sheet001.htm に保存され、メインの html ファイルで使用される URL は「d:/sheet001.htm」またはメインの html ファイルからアクセスできるその他の有効な相対パス.

public string CustomPath { set; }
