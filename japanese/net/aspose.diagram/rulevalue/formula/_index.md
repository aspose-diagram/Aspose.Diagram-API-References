## RuleValue.Formula property

要素の式を表します。この属性には、式がローカルに存在する場合は "someFormula"、式がローカルで削除またはブロックされている場合は "No Formula"、式が継承されている場合は "Inh" のいずれかの文字列を含めることができます。属性が存在しない場合、要素の式は単純な定数です。

public string Formula { get; set; }
