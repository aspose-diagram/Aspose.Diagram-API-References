## PrintSaveOptions.SaveForegroundPagesOnly property

すべてのページを印刷するか、前景のみを印刷するかを指定します。

```csharp
public bool SaveForegroundPagesOnly { get; set; }
```

### 備考

true の場合 - 前景ページ (存在する場合は背景あり) のみを印刷します。 false の場合 - その空の背景ページの後に前景ページ (存在する場合は背景あり) を印刷します。 PageCount &gt; 1 の場合にのみ true を返すことができます。 デフォルト値は false です。 .
