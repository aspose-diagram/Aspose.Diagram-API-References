## ImageSaveOptions.JpegQuality property

生成された JPEG 画像の品質を決定する値を取得または設定します。

```csharp
public int JpegQuality { get; set; }
```

### 備考

JPEG に保存する場合にのみ効果があります。 このプロパティを使用して、JPEG 形式で保存するときに生成された画像の品質を取得または設定します。 値は 0 ～ 100 の範囲で変化する場合があります。0 は品質が最悪ですが最大の圧縮率を意味し、100 は最高を意味します。品質が最小の圧縮. デフォルト値は 95 です.
