[Flags]
public enum PdfPermissions

### 値

| 名前 | 価値 | 説明 |
| --- | --- | --- |
| DisallowAll | `0` | PDF ドキュメントに対するすべての操作を禁止します。これがデフォルト値です。 |
| Printing | `4` | ドキュメントの印刷を許可します。 |
| ModifyContents | `8` | ドキュメントのコンテンツの変更を許可します. |

| FillIn | `100` | フォームへの入力とドキュメントへの署名を許可します。 RC4 40 ビット暗号化を使用する場合、このオプションは無視され、ModifyAnnotations が設定されている場合は常にフォームへの入力が許可されます。 |
| ContentCopyForAccessibility | `200` | 障害のあるユーザーのアクセシビリティをサポートするため、またはその他の目的で、テキストとグラフィックを抽出できるようにします。 RC4 40 ビット暗号化を使用する場合、このオプションは無視され、ContentCopy が設定されている場合はいつでもアクセシビリティが許可されます。 |
| DocumentAssembly | `400` | ドキュメントを組み立てることができます: ページの挿入、回転、または削除、およびブックマークやサムネイル画像などのナビゲーション要素の作成。 RC4 40 ビット暗号化を使用する場合、このオプションは無視され、ModifyContents が設定されている場合はドキュメント アセンブリが許可されます。 |
| HighResolutionPrinting | `804` | ドキュメントを可能な限り最高の解像度で印刷できるようにします。RC4 40 ビット暗号化を使用する場合、このオプションは無視され、印刷が設定されている場合は高解像度印刷が許可されます。 |
| AllowAll | `FFFF` | PDF ドキュメントに対するすべての操作を許可します。 |
