## HTMLSaveOptions.SaveAsSingleFile property

html を単一ファイルとして保存するかどうかを示します。 デフォルト値は false です。

```csharp
public bool SaveAsSingleFile { get; set; }
```

### 備考

複数のページがある場合、それらのページと他のリソースは別のファイルに保存する必要があります。 いくつかのシナリオでは、ユーザーは、転送の利便性などのために結果ファイルを 1 つだけ取得する必要がある場合があります. その場合、ユーザーはこのプロパティを true に設定できます.
