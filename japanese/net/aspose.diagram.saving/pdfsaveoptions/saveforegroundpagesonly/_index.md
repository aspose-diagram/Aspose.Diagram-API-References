## PdfSaveOptions.SaveForegroundPagesOnly property

すべてのページを画像で保存するか、前景のみで保存するかを指定します。

```csharp
public bool SaveForegroundPagesOnly { get; set; }
```

### 備考

true の場合 - 前景ページ (存在する場合は背景あり) のみをレンダリングします。 false の場合 - その空の背景ページの後に前景ページ (存在する場合は背景あり) をレンダリングします。 PageCount &gt; 1 の場合にのみ true を返すことができます。 デフォルト値は false です.
