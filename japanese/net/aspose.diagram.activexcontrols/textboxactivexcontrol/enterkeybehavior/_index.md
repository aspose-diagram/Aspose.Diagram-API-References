## TextBoxActiveXControl.EnterKeyBehavior property

Enter キーの動作を指定します。 True を指定すると、Enter キーを押すと新しい行が作成されます。 False を指定すると、Enter キーを押すとフォーカスがタブ オーダーの次のオブジェクトに移動します。

public bool EnterKeyBehavior { get; set; }
