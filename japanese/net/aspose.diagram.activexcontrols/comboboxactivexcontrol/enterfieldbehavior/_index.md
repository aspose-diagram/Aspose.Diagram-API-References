## ComboBoxActiveXControl.EnterFieldBehavior property

コントロールに入るときの選択動作を指定します。 True は、コントロールが最後にアクティブだったときから選択が変更されないことを指定します。 False は、コントロールに入ったときにコントロール内のすべてのテキストが選択されることを指定します.

public bool EnterFieldBehavior { get; set; }
