## ForeignData.CompressionLevel property

Este atributo solo es significativo si los datos externos son un objeto externo basado en ráster, como un archivo DIB, JPG, PNG, TIFF o GIF. El valor indica el nivel de compresión aplicado al archivo. El nivel de compresión se mide en cientos de un porcentaje.

public double CompressionLevel { get; set; }
