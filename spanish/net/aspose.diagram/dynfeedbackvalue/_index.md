## DynFeedbackValue enumeration

Especifica el tipo de información visual proporcionada a los usuarios cuando arrastran un conector. Cuando se suelta el botón del mouse, la forma del conector resultante no se ve afectada por esta configuración. Este elemento no se aplica a los conectores enrutables.

```csharp
public enum DynFeedbackValue
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| RemainStraight | `0` | Mantente derecho (sin piernas). |
| ShowThreeLegs | `1` | Mostrar tres patas cuando se arrastra. |
| ShowFiveLegs | `2` | Mostrar cinco patas cuando se arrastra. |
| Undefined | `-2147483648` | Indefinido. |
