## EventItem.Action property

Especifica el código de acción del elemento principal EventItem. Para que un elemento EventItem se guarde en un archivo DatadiagramML, debe ser persistente. Actualmente, el único código de acción válido que puede tener un evento persistente es 1 (ONEVENT_ACT_RUNADDON).
