## GluedShapesFlags enumeration

Especifica constantes que identifican qué formas devolver, en función de la dimensionalidad y la direccionalidad de los puntos de conexión que se adhieren a una forma particular; pasado al método Shape.GluedShapes.

```csharp
public enum GluedShapesFlags
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| GluedShapesAll1D | `0` | ID de devolución de todas las formas 1-D que están pegadas a esta forma. |
| GluedShapesIncoming1D | `1` | ID de devolución de formas 1-D cuyos extremos están pegados a esta forma. |
| GluedShapesOutgoing1D | `2` | ID de devolución de formas 1-D cuyos puntos de inicio están pegados a esta forma. |
| GluedShapesAll2D | `3` | Devuelve todas las formas 2D que están pegadas a esta forma y todas las formas 2D a las que está pegada esta forma. |
| GluedShapesIncoming2D | `4` | Si el objeto de origen es una forma 1-D, devuelve los ID de la forma 2-D a la que se pega el punto de inicio. Si el objeto de origen es una forma en 2D, devolver los ID de las formas en 2D que están pegadas a esta forma. |
| GluedShapesOutgoing2D | `5` | Si el objeto de origen es una forma 1-D, devuelve los ID de la forma 2-D a la que se pega el punto final. Si el objeto de origen es una forma en 2D, devolver los ID de las formas en 2D a las que está pegada esta forma. |
