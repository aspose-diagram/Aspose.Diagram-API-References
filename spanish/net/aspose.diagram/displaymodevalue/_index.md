## DisplayModeValue enumeration

Cuando está contenido en un elemento Group, el elemento DisplayMode especifica cómo se muestran una forma de grupo y sus miembros. Cuando está contenido en un elemento SmartTagDef, el elemento DisplayMode determina si la etiqueta inteligente aparece cuando el usuario detiene el mouse sobre la etiqueta, cuando se selecciona la forma, o todo el tiempo.

```csharp
public enum DisplayModeValue
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| HidesShapeText | `0` | Oculta la forma y el texto del grupo. |
| DisplaysShapeBehindMemberShapes | `1` | Muestra la forma del grupo detrás de las formas de los miembros. |
| DisplaysShapeFrontMemberShapes | `2` | Muestra la forma del grupo delante de las formas de los miembros. |
| Undefined | `-2147483648` | Indefinido. |
