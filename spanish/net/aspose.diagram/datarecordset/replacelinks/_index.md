## DataRecordSet.ReplaceLinks property

Define cómo se tratan los enlaces de datos de forma cuando se copian o cortan formas. 1 para reemplazar los vínculos existentes en la forma de destino. 0 para mantener los vínculos existentes en la forma de destino. Si este atributo está ausente, Visio le pregunta al usuario si debe reemplazar los enlaces al copiar o cortar.
