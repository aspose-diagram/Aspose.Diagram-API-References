## RuleValue.Formula property

Representa la fórmula del elemento. Este atributo puede contener una de las siguientes cadenas: "someFormula" si la fórmula existe localmente, "Sin fórmula" si la fórmula se elimina o bloquea localmente, o "Inh" si la fórmula se hereda. Si el atributo no está presente, la fórmula del elemento es una constante simple. Por ejemplo,5 .
