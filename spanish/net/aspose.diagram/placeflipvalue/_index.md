## PlaceFlipValue enumeration

Especifica cómo las formas colocables se voltean y/o giran en una página cuando las formas se distribuyen mediante el comando Diseñar formas en Microsoft Visio. Se permiten los siguientes valores hexadecimales.

public enum PlaceFlipValue
