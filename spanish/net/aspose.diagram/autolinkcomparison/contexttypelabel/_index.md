## AutoLinkComparison.ContextTypeLabel property

Si el valor de ContextType es 2 o 3, este atributo es necesario para definir una comparación. Para ContextType = 2, ContextTypeLabel debe ser la etiqueta del elemento de datos de forma, y si ContextType = 3, ContextTypeLabel debe ser el nombre de la fila local.

```csharp
public string ContextTypeLabel { get; set; }
```
