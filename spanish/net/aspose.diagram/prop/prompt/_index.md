## Prop.Prompt property

El elemento de solicitud especifica el texto descriptivo o instructivo que aparece a los usuarios en el cuadro de diálogo Propiedades personalizadas cuando se selecciona la propiedad. Este texto también aparece como información sobre herramientas cuando el puntero del mouse se detiene sobre la propiedad en la ventana Propiedades personalizadas.
