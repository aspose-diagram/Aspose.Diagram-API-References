## ControlMatchEntryType enumeration

Representa cómo un ListBox o ComboBox busca en su lista a medida que el usuario escribe.

```csharp
public enum ControlMatchEntryType
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| FirstLetter | `0` | El control busca la siguiente entrada que comience con el carácter ingresado. Al escribir repetidamente la misma letra, recorre todas las entradas que comienzan con esa letra. |
| Complete | `1` | A medida que se escribe cada carácter, el control busca una entrada que coincida con todos los caracteres ingresados. |
| None | `2` | La lista no se buscará cuando se escriban caracteres. |
