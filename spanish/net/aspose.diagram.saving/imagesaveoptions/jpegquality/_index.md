## ImageSaveOptions.JpegQuality property

Obtiene o establece un valor que determina la calidad de las imágenes JPEG generadas.

```csharp
public int JpegQuality { get; set; }
```

### Observaciones

Solo tiene efecto cuando se guarda en formato JPEG. Use esta propiedad para obtener o establecer la calidad de las imágenes generadas cuando se guardan en formato JPEG. El valor puede variar de 0 a 100, donde 0 significa la peor calidad pero máxima compresión y 100 significa la mejor calidad pero compresión mínima. El valor por defecto es 95.
