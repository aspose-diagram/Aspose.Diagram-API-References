## PdfPermissions enumeration

Especifica los permisos de usuario para el documento PDF.

```csharp
[Flags]
public enum PdfPermissions
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| DisallowAll | `0` | Deshabilita todas las operaciones en el documento PDF. Este es el valor predeterminado. |
| Printing | `4` | Permite imprimir el documento. |
| ModifyContents | `8` | Permite modificar el contenido del documento. |
| ContentCopy | `10` | Permite copiar o extraer texto y gráficos del documento, incluida la extracción con fines de accesibilidad. |
| ModifyAnnotations | `20` | Permite agregar o modificar anotaciones de texto. Al utilizar el cifrado RC4 de 40 bits, esta opción también permite completar los campos del formulario. |
| FillIn | `100` | Permite rellenar formularios y firmar el documento. Cuando se usa el cifrado RC4 de 40 bits, esta opción se ignora y se permite completar el formulario cada vez que se establece ModifyAnnotations. |
| ContentCopyForAccessibility | `200` | Permite extraer texto y gráficos en apoyo de la accesibilidad para usuarios discapacitados o para otros fines. Cuando se usa el cifrado RC4 de 40 bits, esta opción se ignora y se permite la accesibilidad siempre que se establezca ContentCopy. |
| DocumentAssembly | `400` | Permite ensamblar el documento: insertar, rotar o eliminar páginas y crear elementos de navegación como marcadores o imágenes en miniatura. Cuando se utiliza el cifrado RC4 de 40 bits, esta opción se ignora y se permite el ensamblaje de documentos cuando se configura ModifyContents. |
| HighResolutionPrinting | `804` | Permite imprimir el documento a la resolución más alta posible. Cuando se utiliza el cifrado RC4 de 40 bits, esta opción se ignora y se permite la impresión en alta resolución cuando se establece la opción Impresión. |
| AllowAll | `FFFF` | Permite todas las operaciones sobre el documento PDF. |
