## HTMLSaveOptions.SaveAsSingleFile property

Indica si guardar el html como archivo único. El valor predeterminado es false.

```csharp
public bool SaveAsSingleFile { get; set; }
```

### Observaciones

Si hay varias páginas, esas páginas y otros recursos deben guardarse en archivos separados. En algunos escenarios, es posible que el usuario necesite obtener solo un archivo resultante, por ejemplo, para facilitar la transferencia. Si es así, el usuario puede establecer esta propiedad como verdadera.
