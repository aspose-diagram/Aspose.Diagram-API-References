## PrintSaveOptions.SaveForegroundPagesOnly property

Especifica si se imprimirán todas las páginas o solo en primer plano.

```csharp
public bool SaveForegroundPagesOnly { get; set; }
```

### Observaciones

Si es verdadero: se imprimen solo las páginas de primer plano (con fondo, si está presente). Si es falso: se imprimen las páginas de primer plano (con fondo, si está presente) después de las páginas de fondo vacías. Puede devolver verdadero solo cuando PageCount &gt; 1. El valor predeterminado es falso .
