## RenderingSaveOptions.EmfRenderSetting property

Configuración para representar el metarchivo Emf.

```csharp
public EmfRenderSetting EmfRenderSetting { get; set; }
```

### Observaciones

Los metarchivos EMF identificados como "EMF+ Dual" pueden contener tanto registros EMF+ como registros EMF. Se puede usar cualquier tipo de registro para renderizar la imagen, solo registros EMF+ o solo registros EMF. Cuando!:EmfRenderSetting.EmfPlusPrefer está configurado, luego se analizarán los registros EMF+; de lo contrario, solo se analizarán los registros EMF. El valor predeterminado es!:EmfRenderSetting.EmfOnly.
