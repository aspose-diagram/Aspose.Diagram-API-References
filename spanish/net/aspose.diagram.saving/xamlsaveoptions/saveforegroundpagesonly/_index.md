## XAMLSaveOptions.SaveForegroundPagesOnly property

Especifica si todas las páginas se guardarán en imagen o solo en primer plano.

```csharp
public bool SaveForegroundPagesOnly { get; set; }
```

### Observaciones

Si es verdadero: solo se procesan las páginas de primer plano (con fondo, si está presente). Si es falso: se procesan las páginas de primer plano (con fondo, si está presente) después de las páginas de fondo vacías. Puede devolver verdadero solo cuando PageCount &gt; 1. El valor predeterminado es falso .
